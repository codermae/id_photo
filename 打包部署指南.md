# 证件照系统 - 打包部署完整指南

## 概述

本指南介绍如何将PyQt5应用打包成独立的可执行程序(.exe)，用户无需安装Python即可运行。

---

## 方案对比

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| **PyInstaller** | 支持所有平台、易用 | 文件较大 | ✓ **推荐** |
| **cx_Freeze** | 跨平台 | 配置复杂 | Linux/Mac |
| **py2exe** | Windows只用 | 仅支持Windows | Windows专用 |
| **nuitka** | 编译型、性能好 | 复杂度高 | 高性能需求 |

**推荐使用PyInstaller** - 最成熟、最易用、支持最全面

---

## 方案一：使用PyInstaller打包（推荐）

### 1.1 安装PyInstaller

```bash
pip install pyinstaller
```

### 1.2 准备项目

确保项目结构完整：
```
id_photo_system/
├── main.py
├── config/
├── models/
├── controllers/
├── views/
├── utils/
├── resources/
│   ├── icons/
│   ├── qss/
│   └── models/（AI模型文件）
├── data/
└── requirements.txt
```

### 1.3 创建打包脚本

在项目根目录创建 `build.py`：

```python
# build.py - PyInstaller 打包脚本
import PyInstaller.__main__
import sys
import os

# 获取项目路径
project_path = os.path.dirname(os.path.abspath(__file__))

# PyInstaller参数
args = [
    'main.py',
    '--name=证件照采集系统',
    '--windowed',  # 隐藏控制台窗口
    '--onefile',   # 生成单个可执行文件
    '--icon=resources/icons/app.ico',  # 应用图标
    '--add-data=resources:resources',  # 添加资源文件
    '--add-data=config:config',        # 添加配置
    '--hidden-import=PyQt5.sip',       # 隐藏导入
    '--hidden-import=face_recognition',
    '--hidden-import=dlib',
    '--hidden-import=gfpgan',
    '--hidden-import=rembg',
    '--collect-all=tensorflow',
    '--collect-all=torch',
    f'--distpath={project_path}/dist',  # 输出目录
    f'--buildpath={project_path}/build',
    f'--specpath={project_path}',
]

# 运行PyInstaller
PyInstaller.__main__.run(args)
```

### 1.4 执行打包

```bash
python build.py
```

生成的可执行文件在 `dist/证件照采集系统.exe`

## 方案二：创建安装程序（推荐商业版）

### 2.1 使用NSIS创建安装程序

**优点**：专业、可配置、支持卸载

#### 步骤1：安装NSIS

下载：https://nsis.sourceforge.io/

#### 步骤2：创建安装脚本

在项目根目录创建 `installer.nsi`：

```nsis
; installer.nsi - NSIS安装脚本

; 使用现代UI
!include "MUI2.nsh"

; 设置信息
Name "证件照智能采集系统 v1.0.0"
OutFile "ID_Photo_System_Setup.exe"
InstallDir "$PROGRAMFILES\ID_Photo_System"
RequestExecutionLevel admin

; UI设置
!insertmacro MUI_PAGE_WELCOME
!insertmacro MUI_PAGE_DIRECTORY
!insertmacro MUI_PAGE_INSTFILES
!insertmacro MUI_PAGE_FINISH

!insertmacro MUI_LANGUAGE "SimplifiedChinese"

; 安装文件
Section "安装"
  SetOutPath "$INSTDIR"
  File /r "dist\证件照采集系统\*.*"
  
  ; 创建开始菜单快捷方式
  CreateDirectory "$SMPROGRAMS\证件照采集系统"
  CreateShortcut "$SMPROGRAMS\证件照采集系统\证件照采集系统.lnk" "$INSTDIR\证件照采集系统.exe"
  CreateShortcut "$SMPROGRAMS\证件照采集系统\卸载.lnk" "$INSTDIR\uninstall.exe"
  
  ; 创建桌面快捷方式
  CreateShortcut "$DESKTOP\证件照采集系统.lnk" "$INSTDIR\证件照采集系统.exe"
SectionEnd

; 卸载部分
Section "Uninstall"
  Delete "$SMPROGRAMS\证件照采集系统\*.*"
  RMDir "$SMPROGRAMS\证件照采集系统"
  Delete "$DESKTOP\证件照采集系统.lnk"
  RMDir /r "$INSTDIR"
SectionEnd
```

#### 步骤3：构建安装程序

```bash
# 使用NSIS编译
"C:\Program Files (x86)\NSIS\makensis.exe" installer.nsi
```

结果：`ID_Photo_System_Setup.exe`

## 方案三：创建便携版（推荐分发）

不需要安装，解压即用

### 3.1 创建便携版结构

```
ID_Photo_System_Portable/
├── 证件照采集系统.exe    （主程序）
├── data/                （数据目录）
│   ├── database/       （数据库）
│   ├── photos/         （照片存储）
│   └── backups/        （备份）
├── resources/          （资源）
├── README.txt          （使用说明）
└── launch.bat          （启动脚本）
```

### 3.2 创建启动脚本

创建 `launch.bat`：

```batch
@echo off
REM 证件照采集系统启动脚本
REM 检查系统要求
if not exist "证件照采集系统.exe" (
    echo 错误：找不到主程序文件
    pause
    exit /b 1
)

REM 启动应用
start 证件照采集系统.exe

exit /b 0
```

创建 `launch.sh`（Linux/Mac）：

```bash
#!/bin/bash
# 启动脚本
./证件照采集系统
```

### 3.3 打包便携版

```bash
# 创建压缩包
7z a ID_Photo_System_Portable.7z ID_Photo_System_Portable/
# 或使用zip
powershell Compress-Archive -Path ID_Photo_System_Portable -DestinationPath ID_Photo_System_Portable.zip
```

## 详细打包步骤

### 完整的打包流程

```bash
# 1. 安装依赖
pip install -r requirements.txt
pip install pyinstaller nuitka

# 2. 清理旧构建
rmdir /s /q build dist __pycache__

# 3. 创建应用图标（可选）
# 使用在线工具或图像编辑软件创建 app.ico

# 4. 执行打包
python build.py

# 5. 测试可执行文件
dist\证件照采集系统.exe

# 6. 创建安装程序（可选）
"C:\Program Files (x86)\NSIS\makensis.exe" installer.nsi

# 7. 创建便携版
mkdir ID_Photo_System_Portable
copy dist\证件照采集系统.exe ID_Photo_System_Portable\
xcopy resources ID_Photo_System_Portable\resources /E
xcopy data ID_Photo_System_Portable\data /E
copy launch.bat ID_Photo_System_Portable\

# 8. 压缩
7z a ID_Photo_System_Portable.7z ID_Photo_System_Portable/
```

## 优化打包大小

### 4.1 问题分析

PyInstaller打包后文件较大（300MB+），主要原因：
- TensorFlow/PyTorch等AI库庞大
- 包含完整的Python运行时

### 4.2 优化方案

#### 方案A：排除不必要的依赖

```python
# build.py中修改
args = [
    '--collect-all=tensorflow',
    '--exclude-module=torch',  # 如果不用PyTorch
    '--exclude-module=pandas',  # 如果不用pandas
]
```

#### 方案B：只打包必要的AI库

```python
# 创建最小化依赖文件 requirements-minimal.txt
numpy>=1.26.0
opencv-python>=4.5.0
Pillow>=9.0.0
face-detection-tflite>=0.6.0
SQLAlchemy>=1.4.0
rembg>=2.0.0
GFPGAN>=1.3.8
PyQt5>=5.15.0
```

#### 方案C：使用UPX压缩

```bash
# 安装UPX
choco install upx

# 打包时使用UPX压缩
pyinstaller --upx-dir="C:\UPX" main.py
```

### 4.3 优化结果

| 方案 | 大小 | 说明 |
|------|------|------|
| 原始 | 500MB+ | 包含所有依赖 |
| 最小化 | 300MB | 排除不必要库 |
| UPX压缩 | 150-200MB | 进一步压缩 |
| 分离模型 | 100-150MB | 模型单独下载 |

```ini
[GPU]
use_gpu = True
gpu_device_id = 0

[AI]
face_detection_confidence = 0.3
face_recognition_threshold = 0.6
```

### 5.2 修改main.py以支持打包版本

```python
# main.py
import sys
import os

# 判断是否为打包版本
if getattr(sys, 'frozen', False):
    # 打包版本运行时路径
    BASE_DIR = os.path.dirname(sys.executable)
else:
    # 开发版本路径
    BASE_DIR = os.path.dirname(os.path.abspath(__file__))

# 设置资源路径
os.environ['RESOURCE_PATH'] = os.path.join(BASE_DIR, 'resources')
os.environ['DATA_PATH'] = os.path.join(BASE_DIR, 'data')
```

## 错误处理和测试

### 6.1 常见问题

**问题1：缺少DLL文件**

解决方案：
```python
# build.py中添加
args.extend([
    '--collect-submodules=PyQt5',
    '--hidden-import=PyQt5.plugins.platforms.qwindows',
])
```

**问题2：找不到资源文件**

解决方案：
```python
# 在main.py中处理路径
def get_resource_path(resource_name):
    if getattr(sys, 'frozen', False):
        base_path = sys._MEIPASS
    else:
        base_path = os.path.dirname(__file__)
    return os.path.join(base_path, 'resources', resource_name)
```

**问题3：导入模块错误**

解决方案：
```python
# build.py中添加所有隐藏导入
'--hidden-import=face_recognition',
'--hidden-import=dlib',
'--hidden-import=cv2',
'--hidden-import=PIL',
'--hidden-import=rembg',
```

## 快速开始

### 最简单的方案（3步）

```bash
# 1. 安装PyInstaller
pip install pyinstaller

# 2. 打包
pyinstaller --onefile --windowed --icon=icon.ico main.py

# 3. 完成
# 生成文件：dist/main.exe
```

### 完整的方案（5步）

```bash
# 1. 创建build.py（见1.3节）
# 2. 执行打包
python build.py
# 3. 创建installer.nsi（见2.2节）
# 4. 构建安装程序
"C:\Program Files (x86)\NSIS\makensis.exe" installer.nsi
# 5. 测试并发布
```
