# 导出报表功能说明

## 功能概述

优化后的导出报表功能可以生成一个包含8个数据表的Excel文件，提供全面的统计分析数据。

## 导出的数据表

### Sheet 1：总体统计

包含整体概览信息：

| 统计项目 | 统计数值 |
|---------|---------|
| 采集任务 | 任务名称 |
| 统计日期范围 | 开始日期 至 结束日期 |
| 报表生成时间 | 生成时间戳 |
| 用户总数 | 总用户数 |
| 照片总数 | 总照片数 |
| 已完成 | 已完成人数 |
| 待处理 | 待处理人数 |
| 待采集 | 待采集人数 |
| 无记录 | 无记录人数 |
| 采集记录总数 | 总记录数 |
| 完成率 | 完成率百分比 |
| 男性人数 | 男性数量 |
| 女性人数 | 女性数量 |
| 性别比例 | 男女比例 |

### Sheet 2：用户详细列表

每个用户的详细信息：

| 列名 | 说明 |
|-----|------|
| 序号 | 用户序号 |
| 姓名 | 用户姓名 |
| 身份证号 | 18位身份证号 |
| 性别 | 男/女 |
| 民族 | 民族名称 |
| 年龄 | 实际年龄 |
| 地区 | 所在地区 |
| 采集状态 | 已完成/待处理/待采集/无记录 |
| 原始照片数 | 原始照片数量 |
| 处理后照片数 | 处理后照片数量 |
| 采集日期 | 采集日期或"未采集" |

**用途**：
- 查看每个用户的详细信息
- 筛选特定状态的用户
- 导出用户名单

### Sheet 3：状态分布

采集状态的统计分布：

| 采集状态 | 人数 | 占比(%) |
|---------|------|---------|
| 已完成 | 数量 | 百分比 |
| 待处理 | 数量 | 百分比 |
| 待采集 | 数量 | 百分比 |
| 无记录 | 数量 | 百分比 |
| 合计 | 总数 | 100.00 |

**用途**：
- 了解整体采集进度
- 评估工作完成情况
- 识别需要重点关注的状态

### Sheet 4：地区分布

按地区统计用户分布：

| 地区 | 人数 | 占比(%) |
|-----|------|---------|
| 地区1 | 数量 | 百分比 |
| 地区2 | 数量 | 百分比 |
| ... | ... | ... |

**特点**：
- 按人数降序排列
- 自动从地址中提取地区信息
- 显示占比百分比

**用途**：
- 了解用户地域分布
- 分析不同地区的采集情况
- 制定区域性工作计划

### Sheet 5：民族分布

按民族统计用户分布：

| 民族 | 人数 | 占比(%) |
|-----|------|---------|
| 民族1 | 数量 | 百分比 |
| 民族2 | 数量 | 百分比 |
| ... | ... | ... |

**特点**：
- 按人数降序排列
- 显示占比百分比

**用途**：
- 了解民族构成
- 满足统计报表要求
- 分析多样性

### Sheet 6：性别分布

按性别统计用户分布：

| 性别 | 人数 | 占比(%) |
|-----|------|---------|
| 男 | 数量 | 百分比 |
| 女 | 数量 | 百分比 |
| 未知 | 数量 | 百分比 |

**用途**：
- 了解性别比例
- 分析性别分布
- 满足统计要求

### Sheet 7：年龄分布

按年龄段统计用户分布：

| 年龄段 | 人数 | 占比(%) |
|-------|------|---------|
| 0-18 | 数量 | 百分比 |
| 18-25 | 数量 | 百分比 |

| 36-45 | 数量 | 百分比 |

**特点**：
- 只显示有数据的年龄段

- 了解年龄结构
- 分析不同年龄段的分布
- 制定针对性服务

按日期统计每天的采集情况：

| 日期 | 采集记录数 | 已完成 | 待处理 | 待采集 | 完成率(%) |
|-----|-----------|--------|--------|--------|-----------|
| 2026-04-01 | 数量 | 数量 | 数量 | 数量 | 百分比 |
| 2026-04-02 | 数量 | 数量 | 数量 | 数量 | 百分比 |
| ... | ... | ... | ... | ... | ... |

**特点**：
- 覆盖选定的日期范围
- 只显示有采集记录的日期
- 显示每天的完成率

**用途**：
- 追踪每日工作进度
- 发现工作效率变化
- 分析工作高峰和低谷
- 评估工作计划执行情况

## 使用方法

1. 在统计报表页面选择采集任务
2. 设置开始日期和结束日期
3. 点击"导出报表"按钮
4. 选择保存位置和文件名
5. 等待导出完成

## 文件命名

导出的文件自动命名为：`统计报表_YYYYMMDD_HHMMSS.xlsx`

例如：`统计报表_20260425_143026.xlsx`

## 数据说明

### 状态定义

- **已完成**：已拍照并处理完成（status = 'completed'）
- **待处理**：已拍照但未处理（status = 'processing'）
- **待采集**：已读身份证但未拍照（status = 'pending'）
- **无记录**：只有用户信息，未读身份证（无采集记录）

### 完成率计算

完成率 = 已完成人数 / 采集记录总数 × 100%

**注意**：
- 采集记录总数 = 已完成 + 待处理 + 待采集
- 无记录的用户不计入采集记录总数

### 地区提取规则

1. 如果地址包含"省"，提取到"省"为止
2. 如果地址包含"市"，提取到"市"为止
3. 否则取地址前10个字符
4. 如果地址为空，显示"未知"

## 优化内容

相比旧版本，新版本增加了：

1. **用户详细列表**：完整的用户信息表
2. **状态分布表**：四种状态的详细统计
3. **占比百分比**：所有分布表都增加了占比列
4. **年龄分布表**：新增年龄段统计
5. **每日采集统计**：新增每日进度追踪
6. **更详细的总体统计**：增加了更多统计指标
7. **数据排序**：分布表按人数降序排列
8. **更新状态系统**：使用新的三状态系统（pending、processing、completed）

## 数据分析建议

### 1. 进度分析

查看"状态分布"和"每日采集统计"：
- 完成率是否达标
- 每日进度是否稳定
- 是否存在瓶颈

### 2. 人群分析

查看"地区分布"、"民族分布"、"性别分布"、"年龄分布"：
- 用户构成是否合理
- 是否需要针对特定人群优化服务
- 是否存在覆盖盲区

### 3. 问题排查

查看"用户详细列表"：
- 筛选"待处理"和"待采集"状态
- 找出长时间未完成的用户
- 分析原因并采取措施

### 4. 工作评估

查看"每日采集统计"：
- 对比不同日期的工作量
- 评估工作效率
- 优化工作安排

## 技术说明

- 使用 pandas 库生成 Excel 文件
- 使用 openpyxl 引擎写入数据
- 自动计算所有百分比和统计数据
- 支持大数据量导出（数千条记录）

## 注意事项

1. 导出过程中请勿关闭程序
2. 确保有足够的磁盘空间
3. 如果数据量很大，导出可能需要几秒钟
4. 导出的文件可以用 Excel、WPS 等软件打开
5. 建议定期导出报表进行备份和分析
