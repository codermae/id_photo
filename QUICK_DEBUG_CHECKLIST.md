# 快速调试检查清单

## 当提示"用户不属于采集任务"时

### 检查清单

- [ ] **采集任务是否正确选择？**
  - 查看控制台：`[INFO] 已选择采集任务: XXX (ID: Y)`
  - 确认采集任务名称和ID正确

- [ ] **采集任务中是否有用户？**
  - 查看控制台：`[DEBUG] 采集任务中的用户数: X`
  - 如果为0，说明采集任务中没有用户

- [ ] **你要加载的用户是否在采集任务中？**
  - 查看控制台列出的所有用户
  - 找到你要加载的用户的身份证号
  - 如果找不到，说明用户不在该采集任务中

- [ ] **图片文件名是否正确？**
  - 格式应为：`{身份证号}_{时间戳}.jpg`
  - 身份证号应为18位数字
  - 查看控制台：`[DEBUG] 查询身份证号: XXXXXXXXXXXXXXXXXX`

- [ ] **用户是否存在于数据库？**
  - 查看控制台：`[DEBUG] 用户找到: XXX (ID: Y)`
  - 如果显示 `None`，说明用户不存在

### 解决方案

| 问题 | 解决方案 |
|------|--------|
| 采集任务中没有用户 | 在用户管理中将用户添加到该采集任务 |
| 用户在其他采集任务中 | 选择正确的采集任务，或将用户添加到当前采集任务 |
| 用户不存在于数据库 | 在用户管理中添加该用户 |
| 文件名格式错误 | 重命名文件为 `{身份证号}_{时间戳}.jpg` 格式 |

## 控制台输出快速查看

### 采集任务信息
```
[INFO] 已选择采集任务: 身份核验测试 (ID: 5)
[DEBUG] 采集任务中的用户数: 3
```

### 用户识别信息
```
[DEBUG] 查询身份证号: 110101199001011234
[DEBUG] 用户找到: 张三 (ID: 1)
[DEBUG] 用户在采集任务中: True/False
```

### 成功标志
```
[INFO] 用户识别成功: 张三 (身份证: 110101199001011234)
```

### 失败标志
```
[WARNING] 用户 张三 不属于选定的采集任务
[WARNING] 身份证号 XXX 在数据库中不存在
```

## 一键诊断

运行以下步骤快速诊断问题：

1. **启动应用**
   - 打开控制台窗口

2. **选择采集任务**
   - 在下拉框中选择一个采集任务
   - **记录控制台输出**

3. **加载图像**
   - 选择一个图片文件
   - **记录控制台输出**

4. **对比输出**
   - 采集任务中的用户列表中是否有该用户？
   - 用户在采集任务中的标志是 True 还是 False？
