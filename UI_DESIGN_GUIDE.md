# UI 设计指南

## 一、设计系统概览

### 1.1 设计原则
- **简洁性**：减少视觉混乱，突出核心功能
- **一致性**：统一的样式、交互、反馈
- **可用性**：易于理解和使用
- **美观性**：现代化、专业的外观
- **可访问性**：支持键盘导航、高对比度等

### 1.2 设计目标
- 提升用户体验 30-50%
- 提升系统专业度 50-70%
- 降低学习成本 40-60%
- 提升用户满意度 40-60%

## 四、间距系统

### 4.1 基础单位
```
基础单位：8px

间距规范：
- 4px：微调间距（组件内部）
- 8px：紧凑间距（相关元素）
- 12px：标准间距（组件间距）
- 16px：宽松间距（分组间距）
- 24px：大间距（页面分组）
- 32px：超大间距（主要分组）
```

### 4.2 应用场景
```
按钮内部间距：8px (上下) × 12px (左右)
输入框内部间距：8px (上下) × 12px (左右)
卡片内部间距：16px
组件间距：12px
分组间距：16px
页面边距：16px
```

## 五、组件规范

### 5.1 按钮规范

#### 按钮类型
```
主按钮（Primary）
- 背景：#0078D4
- 文字：白色
- 用途：主要操作（拍照、处理、导出）

次按钮（Secondary）
- 背景：#F3F3F3
- 文字：#323130
- 边框：#D0D0D0
- 用途：次要操作（刷新、取消）

成功按钮（Success）
- 背景：#50E3C2
- 文字：白色
- 用途：确认操作（保存、完成）

危险按钮（Danger）
- 背景：#E81123
- 文字：白色
- 用途：删除操作（删除、清空）

文字按钮（Text）
- 背景：透明
- 文字：#0078D4
- 用途：轻量级操作（更多、帮助）
```

#### 按钮大小
```
小按钮（Small）：32px 高，12px 左右内边距
中按钮（Medium）：40px 高，16px 左右内边距
大按钮（Large）：48px 高，20px 左右内边距
```

#### 按钮状态
```
正常：完整颜色
悬停：颜色加深 10%
按下：颜色加深 20%
禁用：灰色 (#D0D0D0) + 50% 透明度
加载：显示加载动画 + 禁用状态
```

#### 按钮圆角
```
圆角半径：4px
```

### 5.2 输入框规范

#### 输入框样式
```
边框：1px #D0D0D0
背景：#FFFFFF
高度：32px
内边距：8px (上下) × 12px (左右)
圆角：4px
字体：11pt, Regular
```

#### 输入框状态
```
正常：灰色边框

错误：红色边框 (#E81123)
禁用：灰色背景 (#F3F3F3) + 50% 透明度
```

## 六、交互规范

### 6.1 反馈机制

#### 视觉反馈
```
按钮点击：
- 按下：颜色加深 20%
- 释放：恢复原色
- 加载：显示加载动画

输入框焦点：
- 边框变蓝
- 显示光标
- 显示提示文字

表格行选中：
- 背景变蓝
- 文字变白
- 显示选中状态
```

#### 声音反馈
```
操作成功：短促的"叮"声
操作失败：警告音
采集完成：提示音
```

#### 触觉反馈
```
按钮点击：轻微振动
操作完成：短振动
错误提示：长振动
```

### 6.2 加载状态

#### 加载动画
```
旋转圆形：
- 大小：24px
- 颜色：#0078D4
- 速度：1 秒/圈

进度条：
- 显示百分比
- 显示预计时间
- 支持暂停/取消
```

#### 加载提示
```
加载中...
处理中... (50%)
预计剩余时间：2 分钟
```

### 6.3 错误处理

#### 错误提示
```
错误消息框：
- 背景：#FDE7E9
- 边框：1px #E81123
- 图标：红色叉号
- 文字：#E81123

错误信息：
- 清晰说明问题
- 提供解决方案
- 显示错误代码（可选）
```

#### 验证反馈
```
必填字段：
- 标签后显示红色 * 号
- 输入框边框显示红色

验证错误：
- 显示在输入框下方
- 红色文字
- 显示错误原因
```

## 八、响应式设计

### 8.1 断点设置
```
小屏幕（< 1024px）：
- 隐藏非必要元素
- 简化布局
- 增大触摸目标

中屏幕（1024px - 1440px）：
- 标准布局
- 所有元素可见

大屏幕（> 1440px）：
- 宽松布局
- 增加间距
- 显示更多信息
```

### 8.2 适配策略
```
表格：
- 小屏幕：隐藏非关键列
- 中屏幕：显示所有列
- 大屏幕：增加列宽

对话框：
- 小屏幕：全屏显示
- 中屏幕：600px × 400px
- 大屏幕：800px × 600px

按钮：
- 小屏幕：全宽显示
- 中屏幕：自适应宽度
- 大屏幕：固定宽度
```

## 九、无障碍设计

### 9.1 键盘导航
```
Tab：切换焦点
Shift+Tab：反向切换焦点
Enter：激活按钮
Space：切换复选框
Escape：关闭对话框
```

### 9.2 屏幕阅读器
```
所有按钮：添加 aria-label
所有图标：添加 alt 文本
表单字段：关联 label
错误提示：使用 aria-live
```

### 9.3 颜色对比度
```
正常文字：最小对比度 4.5:1
大文字：最小对比度 3:1
UI 组件：最小对比度 3:1
```

## 十一、样式表示例

### 11.1 PyQt5 样式表
```python
# 主色定义
PRIMARY_COLOR = "#0078D4"
SUCCESS_COLOR = "#50E3C2"
WARNING_COLOR = "#FFB81C"
ERROR_COLOR = "#E81123"
NEUTRAL_COLOR = "#F3F3F3"

# 按钮样式
BUTTON_STYLE = """
    QPushButton {
        background-color: """ + PRIMARY_COLOR + """;
        color: white;
        border: none;
        border-radius: 4px;
        padding: 8px 16px;
        font-size: 11pt;
        font-weight: bold;
    }
    QPushButton:hover {
        background-color: #005A9E;
    }
    QPushButton:pressed {
        background-color: #003D7A;
    }
    QPushButton:disabled {
        background-color: #D0D0D0;
        color: #999999;
    }
"""

# 输入框样式
INPUT_STYLE = """
    QLineEdit {
        border: 1px solid #D0D0D0;
        border-radius: 4px;
        padding: 8px 12px;
        background-color: white;
        font-size: 11pt;
    }
    QLineEdit:focus {
        border: 1px solid """ + PRIMARY_COLOR + """;
        background-color: #E7F1F8;
    }
"""

# 对话框样式
DIALOG_STYLE = """
    QDialog {
        background-color: white;
    }
    QGroupBox {
        border: 1px solid #D0D0D0;
        border-radius: 4px;
        margin-top: 8px;
        padding-top: 8px;
        font-weight: bold;
    }
    QGroupBox::title {
        subcontrol-origin: margin;
        left: 10px;
        padding: 0 3px 0 3px;
    }
"""
```
