# 背景颜色深度优化报告

## 优化目标
移除所有重复和相似的背景颜色，保留差异明显且必要的颜色。

## 优化过程

### 阶段1: 移除完全重复的颜色
**从 17种 → 10种**

移除的颜色（RGB值完全相同）:
- 印度签证白 (255, 255, 255) → 使用"白色"
- 澳洲签证白 (255, 255, 255) → 使用"白色"
- 教师资格证白 (255, 255, 255) → 使用"白色"
- 公务员考试蓝 (67, 142, 219) → 使用"蓝色"
- 加拿大签证白 (248, 248, 248) → 使用"日本护照白"
- 会计证蓝 (70, 130, 180) → 使用"英国签证蓝"
- 医师证白 (248, 248, 248) → 使用"日本护照白"

### 阶段2: 移除相似的颜色
**从 10种 → 8种**

移除的颜色（RGB距离 < 25）:
- 英国签证蓝 (70, 130, 180) → 与"美国护照蓝"距离20.83，保留美国护照蓝
- 新加坡护照白 (250, 250, 250) → 与"日本护照白"距离3.46，保留日本护照白

### 阶段3: 最终精简
**从 8种 → 7种**

移除的颜色（与多个颜色都相似）:
- 日本护照白 (248, 248, 248) → 与"白色"距离12.12，与"欧盟护照灰"距离13.86
  - 日本等国可以使用"白色"(255,255,255)
  - 需要浅色的可以使用"欧盟护照灰"(240,240,240)

## 技术细节

### 颜色距离计算
使用欧氏距离公式：
```
distance = √[(R₁-R₂)² + (G₁-G₂)² + (B₁-B₂)²]
```

### 相似度阈值
- < 10: 几乎相同，必须合并
- 10-30: 相似，建议合并
- 30-50: 接近，可以保留
- > 50: 明显不同，应该保留

### 最终配置
所有颜色对距离 ≥ 26，满足"可以保留"标准。
