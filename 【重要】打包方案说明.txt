================================================================================
【场景1】我想立即生成安装程序，最简单最快
================================================================================

✅ 选择脚本：pack.py

执行方式：
  python pack.py

耗时：5-8分钟

生成文件：
  dist/证件照采集系统.exe         (绿色版)
  dist/证件照采集系统_Setup.exe   (安装程序) ← 最终产品
  dist/README.txt                (用户文档)

特点：
  • 最简洁明了
  • 自动完成所有步骤
  • 内置错误处理
  • 推荐使用！

================================================================================
【场景2】我想要完整的打包工具，有更多控制选项
================================================================================

✅ 选择脚本：build_installer.py

执行方式：
  python build_installer.py

耗时：8-10分钟

生成文件：
  dist/证件照采集系统.exe
  dist/证件照采集系统_Setup.exe
  dist/启动.bat
  dist/README.txt
  dist/UNINSTALL.txt
  DEPLOYMENT_SUMMARY.txt          (详细总结)

特点：
  • 功能最完整
  • 生成详细的部署说明
  • 包含卸载指南
  • 适合企业级应用

================================================================================
【场景3】我只想要绿色版可执行文件，不要安装程序
================================================================================

✅ 选择脚本：build_simple.py

执行方式：
  python build_simple.py

耗时：2-3分钟

生成文件：
  dist/证件照采集系统.exe         (绿色版)
  dist/启动.bat                   (启动脚本)

特点：
  • 最快速
  • 生成文件最小
  • 用户可直接运行
  • 可在U盘中使用
  • 不需要NSIS

================================================================================
【场景4】我想要完整版，包含所有库和功能

🚀 使用步骤
================================================================================

【第1步】打开命令行

在项目目录 (D:\1BYSJ\id_photo_system) 中：
  Shift + 右键 → 在此处打开PowerShell

【第2步】执行命令

推荐方式：
  python pack.py

或根据需要选择其他脚本

【第3步】等待完成

脚本会打印进度信息，耐心等待即可

【第4步】检查结果

打开 dist/ 文件夹，查看生成的文件

【第5步】分发给用户

根据情况选择分发方式：
  • 仅分发 Setup.exe (推荐)
  • 或分发整个 dist/ 文件夹

================================================================================
📦 三种分发方式
================================================================================

【方式1】仅发送安装程序（最简洁）⭐ 推荐

  发送文件：证件照采集系统_Setup.exe (80-100MB)
  
  用户体验：
    1. 双击安装
    2. 按照向导操作
    3. 自动创建快捷方式
    4. 即可使用
  
  优点：
    • 文件单一
    • 专业清晰
    • 易于管理
  
  缺点：
    • 需要NSIS编译（本地已支持）

【方式2】发送绿色版本（最灵活）

  发送文件：证件照采集系统.exe (100-150MB)
  
  用户体验：
    1. 解压
    2. 直接运行 .exe
    3. 无需安装
  
  优点：
    • 简单快速
    • 可在U盘使用
    • 无需管理员权限
  
  缺点：
    • 无法卸载管理
    • 无开始菜单快捷方式

【方式3】发送整个文件夹（最全面）

  发送文件：dist/ 文件夹 (压缩为 .zip)
  
  用户体验：
    1. 解压
    2. 可选择：
       • 运行 Setup.exe 安装
       • 直接运行 .exe 使用
  
  优点：
    • 给用户选择
    • 包含所有选项
  
  缺点：
    • 文件较大

💬 常见问题
================================================================================

Q: 我应该选哪个脚本？
A: 选 pack.py，最简单最快！

Q: 需要安装NSIS吗？
A: 推荐装。但即使不装，也能用绿色版本 .exe。

Q: 打包后用户怎么安装？
A: 用户双击 Setup.exe，按照向导操作。

Q: 安装程序大小多大？
A: 约 80-100MB（取决于库）

Q: 用户安装后数据保存在哪？
A: %APPDATA%\证件照采集系统\data

Q: 能否跳过某些步骤？
A: 可以，修改脚本中的代码。

Q: 如何修改应用名称？
A: 修改脚本中的 PROJECT_NAME 变量。

Q: 能在 Mac/Linux 上运行吗？
A: 不行，这是 Windows 安装程序。

Q: 如何升级版本？
A: 修改 VERSION 变量，重新运行脚本。
