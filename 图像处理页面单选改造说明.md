# 图像处理页面单选改造说明

## 修改内容

将图像处理页面（非多规格批量页面）的规格和背景色选择从复选框改为单选按钮，确保用户只能选择一个规格和一个背景色。

## 修改原因

原来的设计使用复选框（`QCheckBox`），允许用户选择多个规格和多个背景色。但在正常的单张图像处理流程中，应该只处理一个规格和一个背景色，多选会导致逻辑混乱。

## 修改详情

### 1. 导入单选按钮组件

```python
from PyQt5.QtWidgets import (..., QRadioButton, QButtonGroup, ...)
```

### 2. 规格选择改为单选

#### 修改前（复选框）

```python
# 规格选择（紧凑版）
spec_group = QGroupBox("规格")

# 快速选择按钮（全选、常用、清空）
...

# 规格复选框
self.spec_checkboxes = {}
for size, specs in size_groups.items():
    checkbox = QCheckBox(display_name)
    checkbox.setChecked(specs[0] == "一寸")  # 默认选中一寸
    self.spec_checkboxes[size_key] = checkbox
```

#### 修改后（单选按钮）

```python
# 规格选择（紧凑版 - 单选）
spec_group = QGroupBox("规格（单选）")

# 规格单选按钮
self.spec_radio_buttons = {}
self.spec_button_group = QButtonGroup(self)  # 按钮组实现单选

for size, specs in size_groups.items():
    radio_button = QRadioButton(display_name)
    radio_button.setChecked(specs[0] == "一寸")  # 默认选中一寸
    self.spec_radio_buttons[size_key] = radio_button
    self.spec_button_group.addButton(radio_button)
```

**关键变化**：
- `QCheckBox` → `QRadioButton`
- `spec_checkboxes` → `spec_radio_buttons`
- 添加 `QButtonGroup` 管理单选行为
- 移除快速选择按钮（全选、常用、清空）

### 3. 背景色选择改为单选

#### 修改前（复选框）

```python
# 背景色选择（紧凑版）
bg_group = QGroupBox("背景色")

# 快速选择按钮（全选、常用、清空）
...

# 背景色复选框
self.bg_checkboxes = {}
for color_name in BACKGROUND_COLORS.keys():
    checkbox = QCheckBox(color_name)
    checkbox.setChecked(color_name == '灰色')  # 默认选中灰色
    self.bg_checkboxes[color_name] = checkbox
```

#### 修改后（单选按钮）

```python
# 背景色选择（紧凑版 - 单选）
bg_group = QGroupBox("背景色（单选）")

# 背景色单选按钮
self.bg_radio_buttons = {}
self.bg_button_group = QButtonGroup(self)  # 按钮组实现单选

for color_name in BACKGROUND_COLORS.keys():
    radio_button = QRadioButton(color_name)
    radio_button.setChecked(color_name == '灰色')  # 默认选中灰色
    self.bg_radio_buttons[color_name] = radio_button
    self.bg_button_group.addButton(radio_button)
```

**关键变化**：
- `QCheckBox` → `QRadioButton`
- `bg_checkboxes` → `bg_radio_buttons`
- 添加 `QButtonGroup` 管理单选行为
- 移除快速选择按钮（全选、常用、清空）

### 4. 高级背景效果保持复选框

高级背景效果（渐变、纹理、虚化）保持使用复选框，因为这些效果可以同时选择多个：

```python
# 高级背景选项（保持复选框，可多选）
advanced_bg_label = QLabel("高级背景效果（可多选）:")

self.gradient_check = QCheckBox("渐变")
self.texture_check = QCheckBox("纹理")
self.blur_check = QCheckBox("虚化")
```

### 5. 更新获取选中项的代码

#### 修改前（获取多选）

```python
# 获取选中的规格和颜色
selected_specs = [cb.spec_name for cb in self.spec_checkboxes.values() if cb.isChecked()]
selected_colors = [name for name, cb in self.bg_checkboxes.items() if cb.isChecked()]

if not selected_specs:
    QMessageBox.warning(self, "警告", "请至少选择一个规格")
    return

if not selected_colors:
    QMessageBox.warning(self, "警告", "请至少选择一个背景色")
    return

# 如果选择了多个，跳转到多规格处理
if len(selected_specs) > 1 or len(selected_colors) > 1:
    self.generate_multi_spec(selected_specs, selected_colors)
    return

spec = selected_specs[0]
bg_color = selected_colors[0]
```

#### 修改后（获取单选）

```python
# 获取选中的规格和颜色（单选）
selected_spec = None
for rb in self.spec_radio_buttons.values():
    if rb.isChecked():
        selected_spec = rb.spec_name
        break

selected_color = None
for name, rb in self.bg_radio_buttons.items():
    if rb.isChecked():
        selected_color = name
        break

if not selected_spec:
    QMessageBox.warning(self, "警告", "请选择一个规格")
    return

if not selected_color:
    QMessageBox.warning(self, "警告", "请选择一个背景色")
    return

spec = selected_spec
bg_color = selected_color
```

**关键变化**：
- 不再使用列表推导式获取多个选中项
- 直接遍历找到选中的单个项
- 移除多选判断逻辑
- 简化变量命名（`selected_specs` → `selected_spec`）

### 6. 删除不需要的方法

删除了以下快速选择相关的方法：

```python
# 删除的方法
def select_all_specs(self):      # 全选规格
def select_common_specs(self):   # 选择常用规格
def clear_specs(self):           # 清空规格选择
def select_all_colors(self):     # 全选颜色
def select_basic_colors(self):   # 选择基础颜色
def clear_colors(self):          # 清空颜色选择
```

**原因**：单选按钮不需要全选和清空功能。

## 功能影响

### 1. 单张处理流程

**修改前**：
1. 可以选择多个规格和多个背景色
2. 如果选择多个，自动跳转到多规格处理
3. 如果只选择一个，进行单张处理

**修改后**：
1. 只能选择一个规格和一个背景色
2. 直接进行单张处理
3. 逻辑更清晰，不会混淆

### 2. 多规格处理

多规格批量处理功能不受影响，仍然通过"多规格批量生成"按钮打开独立的对话框进行。

### 3. 默认选择

- **规格**：默认选中"一寸"
- **背景色**：默认选中"灰色"
- **高级背景效果**：默认全部不选

### 4. 保存功能

保存处理后照片时，使用选中的单个规格和单个背景色，不再需要判断多选情况。

## 优点

### 1. 逻辑清晰

- 单张处理就是单规格单颜色
- 多规格处理通过专门的批量对话框
- 不会混淆两种处理模式

### 2. 用户体验好

- 单选按钮更符合用户预期
- 不会误选多个导致意外行为
- 界面更简洁（移除了快速选择按钮）

### 3. 代码简化

- 不需要判断选择数量
- 不需要处理多选逻辑
- 减少了6个不必要的方法

### 4. 避免错误

- 防止用户误选多个规格
- 防止用户误选多个背景色
- 减少了用户操作错误的可能性

## 注意事项

### 1. 必须选择

用户必须选择一个规格和一个背景色才能进行处理，否则会提示警告。

### 2. 默认选择

系统会默认选中"一寸"规格和"灰色"背景，用户可以更改。

### 3. 高级背景效果

高级背景效果（渐变、纹理、虚化）仍然可以多选，因为这些效果可以叠加使用。

### 4. 多规格需求

如果需要生成多个规格或多个背景色的照片，请使用"多规格批量生成"按钮。
