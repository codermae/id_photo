# 🚀 智能批量处理 - 快速参考

## 打开方式
```
图像处理界面 → [🚀 智能批量处理]
```

## 两种模式

### 📋 单张多规格生成
**一张图片 → 多种规格和颜色**

```
1. 选择规格 (勾选需要的)
   [全选] [常用] [国际]
   
2. 选择颜色 (勾选需要的)
   白色、蓝色、红色、灰色...
   
3. 配置选项
   模式: 精细/高保真
   美颜: 磨皮、祛痘
   
4. 点击 [▶️ 开始处理]
```

**推荐配置**: 4规格 × 4颜色 = 16张 (~48秒)

### 📦 多张批量处理
**多张图片 → 批量处理**

```
1. 选择图片
   [选择图片文件] 或 [选择文件夹]
   
2. 选择模式
   ☑ 单规格模式 (每张1种规格)
   ☐ 多规格模式 (每张多种规格)
   
3. 配置参数
   规格、颜色、背景模式、美颜...
   
4. 选择输出目录
   [选择]
   
5. 点击 [▶️ 开始处理]
```

## 处理选项

### 背景模式
- **精细模式**: 最佳边缘质量，处理时间较长
- **高保真模式**: 商业级质感，需要更多时间

### 美颜选项
- **磨皮**: 平滑皮肤纹理
- **祛痘**: 去除皮肤瑕疵
- **Alpha Matte**: 提升头发丝等边缘细节

### 亮度调整
- 范围: -100 ~ +100
- 推荐: +10 (轻微提亮)

## 规格说明
