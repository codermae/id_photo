## 一、磨皮算法详解

### 原理

磨皮算法基于**双边滤波**技术，在平滑皮肤纹理的同时保持边界清晰。双边滤波同时考虑空间距离和像素值差异，因此能够在不破坏轮廓的前提下进行平滑处理。

### 核心参数

磨皮强度（0.0-1.0）直接影响以下参数：

**公式**：
- 滤波核大小：`d = max(9, int(15 + 20 × strength))`
  - 强度0.0时：9像素
  - 强度1.0时：35像素
  - 核越大，平滑范围越大

- 颜色标准差：`σ_color = max(30, int(40 + 60 × strength))`
  - 强度0.0时：30
  - 强度1.0时：100
  - 决定颜色相似度阈值，值越大平滑范围越广

- 空间标准差：`σ_space = max(30, int(40 + 60 × strength))`
  - 与颜色标准差相同
  - 决定空间平滑范围

- 迭代次数：`iterations = max(1, int(1 + strength × 2))`
  - 强度0.0时：1次迭代
  - 强度1.0时：3次迭代
  - 迭代次数越多，平滑效果越明显

### 特征保护机制

为了避免过度磨皮导致面部特征丢失，系统采用**特征保护掩膜**：

**保护区域**：
- 眼睛区域：半径15像素的圆形
- 眉毛区域：椭圆形（20×10像素）
- 嘴巴区域：椭圆形（20×10像素）
- 鼻孔区域：半径8像素的圆形

**工作原理**：
1. 创建人脸区域掩膜
2. 创建特征保护掩膜
3. 皮肤区域 = 人脸区域 - 特征保护区域
4. 仅在皮肤区域进行双边滤波

### 混合合成

最终结果使用加权混合：

**公式**：
```
Result(x, y, c) = Image(x, y, c) × (1 - mask(x, y) × blend_strength)
                 + Smoothed(x, y, c) × mask(x, y) × blend_strength
```

其中：
- mask(x, y) 为皮肤掩膜（0-1）
- blend_strength 为混合强度，上限0.8（避免过度处理）
- 对BGR三个通道分别处理

## 二、祛痘算法详解

### 痘痘检测原理

祛痘使用**形态学操作**检测痘痘（暗点）：

**步骤1：形态学变换**
- TopHat变换 = Image - Opening(Image)
  - 检测亮点
- BlackHat变换 = Closing(Image) - Image
  - 检测暗点（痘痘）
- 合并 = TopHat + BlackHat

**步骤2：核大小**
- 使用椭圆结构元素，大小9×9像素
- 该大小适合检测2-150像素范围的痘痘

### 痘痘过滤条件

**面积过滤**：
```
有效痘痘 = 2 ≤ Area ≤ (50 + 100 × strength)
```
- 最小面积：2像素（太小的噪点）
- 最大面积：50-150像素（取决于强度）

**圆度过滤**：
```
Circularity = (4π × Area) / Perimeter²
Valid = Circularity > 0.3
```
- 痘痘通常是圆形的
- 圆度系数>0.3时认为是痘痘
- 避免误检矩形或细长的瑕疵

### 修复算法

使用**Telea Inpaint算法**进行修复：

**原理**：
- 根据周围像素推断修复区域
- 使用快速行进法（Fast Marching）计算修复
- 修复半径：`radius = int(5 + 5 × strength)`
  - 强度0.0时：5像素
  - 强度1.0时：10像素

**处理流程**：
1. 创建痘痘掩膜
2. 定义修复半径
3. 使用cv2.inpaint进行修复
4. 替换原图中的痘痘区域

## 三、亮度调整详解

### 原理

亮度调整通过改变像素值来增加或减少图像的整体明亮度。

### 调整范围

**范围**：-100 到 +100

**实现方式**：
```python
if brightness > 0:  # 增加亮度
    adjusted = cv2.convertScaleAbs(frame, alpha=1, beta=brightness)
else:  # 降低亮度
    adjusted = cv2.convertScaleAbs(frame, alpha=1 + brightness/100, beta=0)
```

### 对不同亮度的照片的影响

**过暗图像**（亮度< 60）
- 增加亮度 +30：使细节更清晰
- 增加亮度 +50：可能导致过曝失细节

**正常图像**（亮度 80-120）
- 增加亮度 +10：提亮肤色
- 增加亮度 +20：明显变亮

**过亮图像**（亮度> 180）
- 降低亮度 -20：恢复细节
- 降低亮度 -40：明显变暗

### 最佳范围

系统质量评分中的**最佳亮度范围为 100-150**：
- 小于80：评分为50分
- 80-100：评分为85分
- 100-150：评分为100分（最优）
- 150-170：评分为85分
- 大于170：评分为30分

## 四、对比度调整详解

### 原理

对比度是指图像中亮度差异的大小。高对比度使明暗差异明显，低对比度使图像平淡。

对比度通常用**标准差**来衡量：
- 标准差大 = 对比度高
- 标准差小 = 对比度低

### 调整范围

**范围**：-100 到 +100

**实现方式**：
```python
contrast_factor = 1 + contrast / 100
adjusted = cv2.convertScaleAbs(frame, alpha=contrast_factor, beta=0)
```

### 对不同对比度的照片的影响

**低对比度图像**（标准差< 40）
- 图像显得平淡无神
- 人脸细节不突出
- 增加对比度 +20 能明显改善

**正常对比度**（标准差 60-80）
- 最适合证件照
- 人脸细节清晰
- 黑眼睛、嘴巴清晰可见

**高对比度图像**（标准差> 100）
- 明暗差异过大
- 可能出现过曝或欠曝
- 增加对比度 -20 能改善

### 最佳范围

系统质量评分中的**最佳对比度范围为 60-80**：
- 小于40：评分为30分（过低）
- 40-50：评分为50分
- 50-60：评分为70分
- 60-80：评分为100分（最优）
- 80-90：评分为85分
- 90-100：评分为70分
- 大于100：评分为30分（过高）

## 五、综合质量评分计算

### 三维度评分模型

虽然系统检测了9个维度（位置、距离、嘴巴、正视、光线等），但**综合评分只使用3个核心维度**：

**评分权重**：
- 清晰度：40%
- 亮度：30%
- 对比度：30%

**公式**：
```
overall_score = ⌊sharpness×0.4 + brightness×0.3 + contrast×0.3⌋
```

### 清晰度评分（40%权重）

**原理**：使用拉普拉斯算子计算边缘强度
```
Sharpness = min(100, ⌊Laplacian_Variance / 5⌋)
```

**含义**：
- 方差越大 = 边缘越清晰
- 除以5进行缩放，使其适应0-100范围

### 质量等级对应

| 总分范围 | 等级 | 说明 |
|---------|------|------|
| 90-100 | 优秀 | 照片质量完美 |
| 80-89 | 良好 | 照片质量良好 |
| 70-79 | 合格 | 照片质量合格 |
| < 70 | 不合格 | 需要重拍 |

系统默认**要求质量≥70分才允许拍照**。

## 六、高级检测维度（现已集成但未在最终评分中使用）

系统还检测了以下维度，用于**实时反馈**但不参与最终评分：

### 人脸位置检测
- **检测方法**：计算人脸中心与图像中心的偏移
- **容忍度**：水平10%，垂直8%
- **结果**：center（中心）/ left（左）/ right（右）/ up（上）/ down（下）
- **用途**：在实时预览中提示用户调整位置

### 人脸大小检测
- **检测方法**：计算人脸面积占图像面积的比例
- **标准**：最优 5%-30%
- **结果**：too_small（太小）/ too_large（太大）/ good（良好）
- **用途**：提示用户靠近或远离摄像头

### 眼睛正视检测
- **检测方法**：分析眼睛对称性，判断是否正视镜头
- **严格条件**：
  - 左右眼睛对称性差异 < 0.5
  - 平均眼睛分数 ≥ 2.0
  - 任一眼睛分数 ≥ 1.5
- **用途**：提示用户正视镜头

### 嘴巴闭合检测
- **检测方法**：轮廓分析 + 亮度对比 + 中心暗度
- **判断标准**：综合得分≥2分认为张嘴
- **用途**：提示用户闭嘴

### 光照检测
- **检测方法**：
  - 亮度范围：60-180
  - 亮度标准差：>20
- **用途**：评估光线是否充足

## 七、实际应用建议

### 磨皮强度建议
- **0.3-0.5**：自然磨皮，适合日常证件照
- **0.5-0.7**：明显磨皮，适合重要场合
- **0.7-1.0**：强效磨皮，需要谨慎，可能过度

### 祛痘强度建议
- **0.3-0.5**：温和祛痘，保留小的瑕疵
- **0.5-0.7**：标准祛痘，去除明显痘痘
- **0.7-1.0**：强效祛痘，去除所有瑕疵

### 亮度调整建议
- **原始亮度< 80**：建议增加亮度 +15 到 +30
- **原始亮度 80-120**：建议调整 -10 到 +10
- **原始亮度> 150**：建议降低亮度 -20 到 -30

### 对比度调整建议
- **原始对比度< 40**：建议增加对比度 +15 到 +25
- **原始对比度 40-60**：建议增加对比度 +5 到 +15
- **原始对比度 60-100**：建议调整 -5 到 +5
- **原始对比度> 100**：建议降低对比度 -15 到 -25
