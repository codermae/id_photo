# 06 用户界面设计

## 6.1 主窗口设计

**布局**: 标签页(Tab) + 状态栏  
**尺寸**: 1400x900像素  
**主题**: 现代简约风格

**组成部分**:
```
┌────────────────────────────────────────────┐
│  标题栏: 证件照采集及处理系统 v1.0          │
├────────────────────────────────────────────┤
│ [身份证读取][数据导入][拍照采集][图像处理] │
│ [数据管理][统计报表]    [备份][帮助][关于] │
├────────────────────────────────────────────┤
│                                            │
│            当前标签页内容区域               │
│                                            │
├────────────────────────────────────────────┤
│ 状态栏: 用户总数 | 照片总数 | 完成率       │
└────────────────────────────────────────────┘
```

## 6.2 六大功能页面

### 6.2.1 身份证读取页面

**布局**:
```
┌─ 采集任务 ─────────────────────┐
│ 当前任务: [下拉框]              │
└────────────────────────────────┘
┌─ 身份证信息 ───────────────────┐
│ 姓名: [____]  性别: [____]     │
│ 身份证号: [________________]   │
│ 民族: [____]  生日: [____]     │
│ 地址: [____________________]   │
│ [开始读取] [清空] [保存]       │
└────────────────────────────────┘
┌─ 身份证照片 ───────────────────┐
│        [照片显示区域]           │
└────────────────────────────────┘
```

**按钮功能**:
- **开始读取**: 启动读卡器，读取身份证信息
- **清空**: 清空当前显示的信息
- **保存**: 将信息保存到数据库

**操作流程**:
1. 选择采集任务
2. 点击"开始读取"
3. 将身份证放在读卡器上
4. 系统自动读取并显示信息
5. 确认无误后点击"保存"

### 6.2.2 数据导入页面

**布局**:
```
┌─ 采集任务 ─────────────────────┐
│ [新建任务] [选择任务▼]         │
└────────────────────────────────┘
┌─ 导入设置 ─────────────────────┐
│ Excel文件: [选择文件...]       │
│ [下载模板]                     │
│ [开始导入]                     │
└────────────────────────────────┘
┌─ 导入结果 ─────────────────────┐
│ 成功: 0  失败: 0  重复: 0      │
│ [查看详情]                     │
└────────────────────────────────┘
```

**按钮功能**:
- **新建任务**: 创建新的采集任务
- **选择文件**: 选择Excel文件
- **下载模板**: 下载标准Excel模板
- **开始导入**: 执行批量导入
- **查看详情**: 查看导入详细结果

**Excel模板格式**:
| 姓名 | 身份证号 | 性别 | 民族 | 出生日期 | 地址 |
|------|---------|------|------|---------|------|
| 张三 | 110101199001011234 | 男 | 汉族 | 1990-01-01 | 北京市... |

### 6.2.3 拍照采集页面

**布局**:
```
┌─ 用户信息 ─────────────────────┐
│ 当前用户: 张三 (110101...)     │
│ [选择用户▼] [清除]             │
└────────────────────────────────┘
┌─ 摄像头预览 ───────────────────┐
│                                │
│      [实时预览画面]             │
│                                │
│ 质量评分: 85分 [人脸检测✓]     │
└────────────────────────────────┘
┌─ 拍照控制 ─────────────────────┐
│ 摄像头: [选择▼] [刷新]         │
│ [拍照] [重拍] [保存]           │
└────────────────────────────────┘
```

**按钮功能**:
- **选择用户**: 从数据库选择用户
- **清除**: 清除当前用户
- **刷新**: 刷新摄像头列表
- **拍照**: 拍摄照片
- **重拍**: 重新拍摄
- **保存**: 保存照片到数据库

**实时功能**:
- 人脸检测框显示
- 质量评分实时更新
- 语音提示（"请看镜头"、"拍照成功"）

### 6.2.4 图像处理页面

**布局**:
```
┌─ 任务选择 ─────────────────────┐
│ 采集任务: [选择▼]              │
└────────────────────────────────┘
┌─ 用户列表 ─────────────────────┐
│ [搜索框]                       │
│ ☐ 张三  ☐ 李四  ☐ 王五         │
│ [全选] [反选]                  │
└────────────────────────────────┘
┌─ 处理参数 ─────────────────────┐
│ 规格: ☑一寸 ☑二寸 ☑35x45mm    │
│ 背景: ☑白色 ☑蓝色 ☑红色       │
│ 美颜: [滑块 0-100]             │
│ [批量处理] [单张处理]          │
└────────────────────────────────┘
┌─ 预览区域 ─────────────────────┐
│ [原图]  →  [处理后]            │
└────────────────────────────────┘
```

**按钮功能**:
- **全选/反选**: 批量选择用户
- **批量处理**: 处理所有选中用户
- **单张处理**: 处理当前预览的照片

**处理流程**:
1. 选择采集任务
2. 选择要处理的用户
3. 设置处理参数（规格/背景/美颜）
4. 点击"批量处理"
5. 显示进度条
6. 处理完成提示

### 6.2.5 数据管理页面

**布局**:
```
┌─ 采集任务 ─────────────────────┐
│ 当前任务: [全部▼]              │
└────────────────────────────────┘
┌─ 搜索筛选 ─────────────────────┐
│ 关键词: [____] [搜索]          │
│ 性别:[全部▼] 民族:[全部▼]     │
│ 年龄:[0]-[150] 状态:[全部▼]   │
│ [刷新]                         │
└────────────────────────────────┘
┌─ 用户列表 ─────────────────────┐
│ ID│姓名│身份证号│性别│状态│操作│
│ 1 │张三│110101..│男 │已采│查看│
│ 2 │李四│110102..│女 │待采│查看│
│ [删除选中] [导出Excel]         │
└────────────────────────────────┘
```

**按钮功能**:
- **搜索**: 按关键词搜索
- **刷新**: 刷新列表并重置筛选
- **查看**: 查看用户详情
- **删除选中**: 批量删除用户
- **导出Excel**: 导出用户列表

**高级搜索**:
- 支持姓名/身份证号模糊搜索
- 支持性别/民族/年龄/状态筛选
- 支持多条件组合筛选

## 6.3 对话框设计

### 6.3.1 裁剪对话框
- 显示原图
- 拖动调整裁剪区域
- 实时预览裁剪结果
- 确认/取消按钮

### 6.3.2 批量处理对话框
- 进度条显示
- 当前处理文件名
- 成功/失败计数
- 取消按钮

### 6.3.3 备份恢复对话框
- 备份列表
- 备份/恢复按钮
- 删除备份按钮

## 6.4 交互设计

### 6.4.1 反馈机制
- **进度提示**: 长时间操作显示进度条
- **成功提示**: 绿色提示框
- **错误提示**: 红色提示框
- **确认对话框**: 危险操作需确认

### 6.4.2 快捷键
- `Ctrl+S`: 保存
- `Ctrl+N`: 新建
- `Ctrl+F`: 搜索
- `F5`: 刷新
- `Delete`: 删除

### 6.4.3 拖拽支持
- 图片文件拖拽到处理页面
- Excel文件拖拽到导入页面

## 6.5 样式设计

**配色方案**:
- 主色: #0078d4 (蓝色)
- 成功: #5CB87A (绿色)
- 警告: #FAC858 (黄色)
- 错误: #EE6666 (红色)
- 背景: #F5F5F5 (浅灰)

**字体**:
- 标题: 14px 粗体
- 正文: 12px 常规
- 按钮: 12px 粗体
