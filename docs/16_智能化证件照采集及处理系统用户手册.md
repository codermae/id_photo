## 1. 系统概述

### 1.1 系统简介
智能化证件照采集及处理系统是一款基于AI技术的桌面应用软件，专门用于证件照的快速采集、智能处理和规范管理。系统集成了身份证读取、摄像头拍照、AI美颜、背景替换、智能裁剪等功能，适用于学校、企业、政府机构等需要批量制作证件照的场景。

### 1.2 主要特点
- **高效采集**: 支持身份证读取、Excel批量导入、摄像头拍照
- **智能处理**: AI美颜、背景替换、智能裁剪、质量评估
- **规范管理**: 采集任务管理、用户数据管理、照片分类
- **数据分析**: 多维度统计报表、可视化图表
- **用户友好**: 直观的图形界面、语音提示、实时预览

### 1.3 应用场景
- **教育机构**: 新生入学证件照采集
- **企业单位**: 员工证件照制作
- **政府部门**: 身份证照片采集
- **商业服务**: 照相馆、打印店
- **自助设备**: 证件照自助采集终端

## 3. 安装与配置

### 3.1 安装步骤

#### 步骤1：安装Python
1. 访问 https://www.python.org/downloads/
2. 下载Python 3.8+版本
3. 安装时勾选"Add Python to PATH"
4. 验证安装：`python --version`

#### 步骤2：安装依赖
```bash
cd id_photo_system
pip install -r requirements.txt
```

#### 步骤3：下载AI模型
1. 下载GFPGAN模型：https://github.com/TencentARC/GFPGAN/releases
2. 下载文件：GFPGANv1.3.pth
3. 放置位置：`gfpgan/weights/GFPGANv1.3.pth`

#### 步骤4：运行系统
```bash
python main.py
```

### 3.2 首次配置
1. 系统首次运行会自动创建数据库
2. 数据库位置：`data/database/id_photo.db`
3. 照片存储位置：`data/photos/`
4. 备份位置：`data/backups/`

## 4. 快速入门

### 4.1 创建采集任务
1. 启动系统
2. 点击"新建采集任务"
3. 输入任务名称、组织机构、描述
4. 设置开始和结束日期
5. 点击"保存"

### 4.2 导入用户数据
**方法一：身份证读取**
1. 连接身份证读卡器
2. 选择采集任务
3. 点击"开始读取"
4. 放置身份证在读卡器上
5. 系统自动读取并保存信息

**方法二：Excel导入**
1. 下载Excel模板
2. 填写用户信息
3. 选择Excel文件
4. 点击"开始导入"
5. 查看导入结果

### 4.3 拍照采集
1. 选择用户
2. 打开摄像头
3. 调整位置（系统实时提示）
4. 点击"拍照"
5. 系统自动保存照片

### 4.4 图像处理
1. 选择用户照片
2. 设置处理参数：
   - 规格：一寸、二寸等
   - 背景：白色、蓝色、红色等
   - 美颜：强度0-100%
3. 点击"应用处理"
4. 预览效果
5. 点击"保存照片"

## 5. 功能详解

### 5.1 身份证读取模块
**功能**:
- 自动读取二代身份证信息
- 提取姓名、身份证号、性别、民族、生日、地址
- 身份证照片自动生成
- 重复检测和提示

**操作流程**:
1. 连接读卡器硬件
2. 选择采集任务
3. 点击"开始读取"
4. 放置身份证
5. 系统自动读取并显示信息
6. 确认无误后保存

### 5.2 数据导入模块
**支持格式**:
- Excel文件 (.xlsx, .xls)
- CSV文件
- 手动录入

**Excel模板格式**:
| 姓名 | 身份证号 | 性别 | 民族 | 出生日期 | 地址 |
|------|---------|------|------|---------|------|
| 张三 | 110101199001011234 | 男 | 汉族 | 1990-01-01 | 北京市... |

**数据验证**:
- 身份证号格式校验
- 必填字段检查
- 重复身份证号检测
- 数据完整性验证

### 5.3 拍照采集模块
**实时功能**:
- 摄像头预览（30fps）
- 人脸检测和框选
- 质量评分实时显示
- 身份核验（与身份证照片比对）
- 语音提示引导

**拍照条件**:
- 检测到且仅有1个人脸
- 质量评分 > 70分
- 身份核验相似度 > 60%
- 眼睛正视镜头
- 嘴巴闭合

**自动拍照**:
- 启用自动拍照功能
- 当所有条件满足时自动倒计时3秒
- 倒计时期间状态改变自动取消

### 5.4 图像处理模块
**AI美颜功能**:
- **磨皮**: 平滑皮肤纹理
- **祛痘**: 去除痘痘和瑕疵
- **眼部增强**: 提亮眼睛
- **唇部增强**: 增强唇色
- **牙齿美白**: 美白牙齿
- **瘦脸**: 轻微瘦脸
- **大眼**: 放大眼睛
- **鼻部增强**: 增强鼻梁

**背景替换**:
- **纯色背景**: 白色、蓝色、红色、灰色等
- **渐变背景**: 垂直、水平、对角线渐变
- **纹理背景**: 自定义纹理图像
- **虚化背景**: 保留原始背景但虚化

**智能裁剪**:
- **支持规格**: 15种证件照规格
  - 中国标准：一寸、二寸、小二寸、大一寸等
  - 国际标准：美国护照、欧盟护照、日本护照等
  - 特殊规格：驾驶证、社保卡等
- **裁剪算法**: 基于人脸检测的智能定位
- **头顶留白**: 符合证件照标准

### 5.5 批量处理模块
**功能特点**:
- 支持多规格同时生成
- 支持多背景色同时生成
- 多线程并行处理
- 实时进度显示
- 错误处理和重试

**操作流程**:

### 5.6 数据管理模块

## 7. 常见问题

### 7.1 安装问题

#### Q1: 安装依赖时出现错误
**问题**: `pip install -r requirements.txt` 失败
**解决**:
1. 检查Python版本：`python --version`
2. 确保Python 3.8-3.11
3. 尝试单独安装失败包：`pip install package_name`
4. 使用国内镜像源：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple`

#### Q2: GFPGAN模型下载失败
**问题**: 无法下载GFPGANv1.3.pth
**解决**:
1. 手动下载：https://github.com/TencentARC/GFPGAN/releases
2. 创建目录：`gfpgan/weights/`
3. 放置模型文件：`gfpgan/weights/GFPGANv1.3.pth`

### 7.2 运行问题

#### Q1: 摄像头无法打开
**问题**: 摄像头预览黑屏或报错
**解决**:
1. 检查摄像头连接
2. 关闭其他使用摄像头的程序
3. 更新摄像头驱动
4. 在系统设置中检查摄像头权限
5. 尝试其他USB接口

#### Q2: 身份证读卡器无法识别
**问题**: 读卡器状态显示"未连接"
**解决**:
1. 检查USB连接
2. 安装读卡器驱动程序
3. 重启系统
4. 联系读卡器厂商

#### Q3: AI美颜速度慢
**问题**: 美颜处理时间过长
**解决**:
1. 关闭其他占用CPU的程序
2. 降低美颜强度
3. 检查GFPGAN模型是否正确加载
4. 考虑升级硬件

### 7.3 使用问题

#### Q1: 照片质量评分低
**问题**: 系统提示照片质量不合格
**解决**:
1. 调整光照：确保光线均匀
2. 调整位置：人脸居中，正视镜头
3. 调整距离：人脸大小合适
4. 检查表情：自然表情，嘴巴闭合
5. 检查眼镜：避免反光

#### Q2: 背景替换边缘不自然
**问题**: 换背景后边缘有白边或黑边
**解决**:
1. 使用"精细模式"进行背景替换
2. 调整边缘优化参数
3. 确保原始照片背景与人物对比明显
4. 使用更好的照明条件

#### Q3: 批量处理失败
**问题**: 批量处理中途停止或报错
**解决**:
1. 检查输入文件格式
2. 确保有足够磁盘空间
3. 分批处理大量文件
4. 查看错误日志：`logs/error.log`

### 7.4 数据问题

#### Q1: 数据库锁定
**问题**: 系统提示数据库被锁定
**解决**:
1. 关闭其他打开数据库的程序
2. 重启系统
3. 删除数据库锁文件（如有）
4. 恢复最近备份

#### Q2: 数据丢失
**问题**: 用户数据或照片丢失
**解决**:
1. 检查备份文件：`data/backups/`
2. 恢复最近备份
3. 检查照片存储路径
4. 检查文件权限
