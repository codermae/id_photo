# 01 项目概述

## 1.1 项目背景

证件照在现代社会应用广泛（身份证、护照、学生证等），但传统采集方式存在问题：
- 需要专业设备和人员，成本高
- 拍摄效率低，排队时间长
- 照片质量不稳定，返工率高
- 批量采集困难，管理混乱

随着AI技术发展，计算机视觉为智能化证件照系统提供了技术基础。

## 1.2 项目目标

开发智能化证件照采集及处理系统，实现：
- **高效采集**: 身份证读取、摄像头拍照、批量导入
- **智能处理**: AI美颜、背景替换、智能裁剪
- **规范管理**: 采集任务管理、用户数据管理
- **数据分析**: 多维度统计、可视化报表

**性能指标**:
- 单人采集时间 < 30秒
- 单张处理时间 < 3秒
- 人脸检测准确率 > 99%
- 批量处理支持 1000+ 张

## 1.3 核心功能

### 1. 身份证读取
- 自动读取二代身份证信息
- 提取姓名、身份证号、性别、民族、生日、地址
- 身份证号校验、重复检测
- 自动生成身份证照片

### 2. 数据导入
- Excel批量导入用户信息
- 标准模板下载
- 数据完整性校验
- 多任务并行管理

### 3. 拍照采集
- 实时高清预览
- 人脸检测与质量评分
- 语音引导拍照
- 自动保存原始照片

### 4. 图像处理
- GFPGAN AI美颜
- 背景替换（纯色/渐变/自定义）
- 智能裁剪（10种标准规格）
- 批量处理（多规格多背景）

### 5. 数据管理
- 用户信息增删改查
- 高级搜索（姓名/身份证/性别/年龄/民族/状态）
- 照片分类管理
- Excel导出

### 6. 统计报表
- 6种交互式图表（ECharts）
- 多维度统计分析
- 趋势分析
- 数据筛选

## 1.4 技术特色

### AI智能化
- **face-detection-tflite**: 基于BlazeFace的高精度人脸检测（TensorFlow Lite）
- **dlib**: 68点人脸关键点检测
- **GFPGAN**: AI美颜，自然不失真
- **rembg (U2-Net)**: 高精度AI背景分割
- **face-recognition**: 人脸识别与身份核验
- **质量评分**: 多维度照片质量评估

### 高性能
- **多线程**: 批量处理不阻塞UI
- **异步加载**: 图表数据后台处理
- **内存优化**: 大图自动压缩
- **缓存机制**: 重复数据缓存

### 用户友好
- **直观界面**: 标签页设计
- **实时反馈**: 进度条、状态提示
- **语音引导**: 拍照语音提示
- **错误处理**: 友好提示和恢复

## 1.5 创新点

### 技术创新
1. **AI美颜集成**: 首次将GFPGAN应用于证件照
2. **质量评分系统**: 多维度实时质量评估
3. **多线程架构**: 信号槽机制解决Qt线程安全
4. **ECharts报表**: QWebEngineView + QThread架构

### 应用创新
1. **采集任务管理**: 多任务并行、数据隔离
2. **智能地区推断**: 从身份证号推断地区
3. **统一批量处理**: 一键多规格多背景生成

## 1.6 应用场景

- **政府**: 公安、人社、民政系统
- **教育**: 学校、培训机构、考试中心
- **企业**: 员工证、物业管理
- **商业**: 照相馆、打印店、自助设备

| ECharts | 数据可视化 | 交互性强 |
| OpenCV | 图像处理 | 功能全面 |

## 1.8 项目价值

**经济价值**: 降低成本60%，效率提升5倍  
**社会价值**: 便民服务，标准化流程
