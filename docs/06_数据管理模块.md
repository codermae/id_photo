# 数据管理模块

## 模块概述

数据管理模块负责系统中所有数据的存储、查询、统计和导出功能。采用SQLAlchemy ORM框架，使用SQLite数据库，提供完整的用户、照片和采集记录管理功能。

**核心功能**：
- 用户管理：添加、查询、更新、删除用户信息
- 照片管理：记录和管理采集的证件照
- 采集记录：跟踪采集过程和状态
- 数据统计：生成采集统计报表
- 数据导出：支持Excel、CSV等格式导出
- 数据查询：灵活的搜索和筛选功能

#### 用户信息字段

| 字段 | 类型 | 说明 |
|-----|------|------|
| id | Integer | 主键，自增 |
| name | String(50) | 姓名 |

| nation | String(20) | 民族 |
| birthday | Date | 出生日期 |

| id_photo_path | String(255) | 身份证照片路径 |

| face_encoding | LargeBinary | 人脸特征向量（128维） |

| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

### 数据模型

#### User (用户模型)

```python
class User(Base):
    """用户表"""
    __tablename__ = 'users'
    
    id = Column(Integer, primary_key=True)
    name = Column(String(50), nullable=False)
    id_number = Column(String(18), unique=True, nullable=False, index=True)
    gender = Column(String(10))
    nation = Column(String(20))
    birthday = Column(Date)
    address = Column(String(200))
    id_photo_path = Column(String(255))
    id_photo_data = Column(LargeBinary)
    face_encoding = Column(LargeBinary)
    face_encoding_timestamp = Column(DateTime)
    created_at = Column(DateTime, default=datetime.now)
    updated_at = Column(DateTime, default=datetime.now, onupdate=datetime.now)
    
    # 关系
    photos = relationship('Photo', back_populates='user', cascade='all, delete-orphan')
    records = relationship('CollectionRecord', back_populates='user', cascade='all, delete-orphan')
```

#### Photo (照片模型)

```python
class Photo(Base):
    """证件照表"""
    __tablename__ = 'photos'
    
    id = Column(Integer, primary_key=True)
    user_id = Column(Integer, ForeignKey('users.id'), nullable=False, index=True)
    photo_type = Column(String(20), nullable=False)
    file_path = Column(String(255), nullable=False)
    file_size = Column(Integer)
    photo_spec = Column(String(20))
    background_color = Column(String(20))
    quality_score = Column(Integer)
    face_similarity = Column(Integer)
    created_at = Column(DateTime, default=datetime.now, index=True)
    
    # 关系
    user = relationship('User', back_populates='photos')
```

#### CollectionRecord (采集记录模型)

```python
class CollectionRecord(Base):
    """采集记录表"""
    __tablename__ = 'collection_records'
    
    id = Column(Integer, primary_key=True)
    user_id = Column(Integer, ForeignKey('users.id'), nullable=False, index=True)
    collection_date = Column(Date, default=datetime.now, index=True)
    operator = Column(String(50))
    status = Column(String(20), default='pending')
    notes = Column(Text)
    created_at = Column(DateTime, default=datetime.now)
    
    # 关系
    user = relationship('User', back_populates='records')
```

### 数据库配置

# 数据库URL
DATABASE_URL = 'sqlite:///./data/database/id_photo.db'

# 创建引擎
engine = create_engine(DATABASE_URL, echo=False)

# 创建会话工厂
SessionLocal = sessionmaker(autocommit=False, autoflush=False, bind=engine)

# 创建基类
Base = declarative_base()

# 初始化数据库
def init_db():
    Base.metadata.create_all(bind=engine)

## 常见问题

### Q1: 如何处理重复的身份证号？

**A**: 系统已设置身份证号为唯一约束，添加重复身份证号会抛出异常。处理方法：
```python
try:
    user = db_helper.add_user(...)
except IntegrityError:
    # 身份证号已存在
    user = db_helper.get_user_by_id_number(id_number)
```

### Q2: 如何批量导入用户数据？

**A**: 使用批量导入功能：
```python
users_data = [
    {'name': '张三', 'id_number': '110101199001011234', ...},
    {'name': '李四', 'id_number': '110101199001011235', ...},
]
for user_data in users_data:
    db_helper.add_user(**user_data)
```

### Q3: 如何导出特定日期范围的数据？

**A**: 使用日期范围参数：
```python
db_helper.export_collection_stats_to_excel(
    output_path='stats.xlsx',
    start_date=date(2026, 3, 1),
    end_date=date(2026, 3, 31)
)
```
