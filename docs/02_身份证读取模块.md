# 身份证读取模块

## 模块概述

身份证读取模块是系统的数据入口模块，负责通过身份证读卡器自动读取用户的身份信息，并保存到数据库中。该模块支持真实读卡器和测试模式两种工作方式。

---

## 功能特性

### 1. 自动读取

- 连接身份证读卡器
- 自动读取身份证芯片信息
- 提取文字信息（姓名、身份证号、性别、民族、地址等）
- 提取照片信息（身份证照片）

### 2. 数据验证

- 身份证号码格式验证
- 校验码验证（GB 11643-1999标准）
- 重复数据检测
- 数据完整性检查

### 3. 自动保存

- 自动保存用户信息到数据库
- 自动保存身份证照片
- 自动创建采集记录
- 自动更新系统状态

### 4. 测试模式

- 无需真实读卡器
- 自动填充测试数据
- 用于系统演示和测试
- 支持多个测试用户

## 身份证校验算法

### GB 11643-1999标准

中国居民身份证号码采用18位编码：

```
6位地区码 + 8位出生日期 + 3位顺序码 + 1位校验码
```

### 校验码计算

#### 1. 加权因子

```python
WEIGHT_FACTORS = [7, 9, 10, 5, 8, 4, 2, 1, 6, 3, 7, 9, 10, 5, 8, 4, 2]
```

#### 2. 校验码表

```python
CHECK_CODES = ['1', '0', 'X', '9', '8', '7', '6', '5', '4', '3', '2']
```

#### 3. 计算步骤

```python
def calculate_checksum(id_number: str) -> str:
    """
    计算身份证校验码
    
    Args:
        id_number: 前17位身份证号
        
    Returns:
        str: 校验码（0-9或X）
    """
    # 1. 前17位数字与加权因子相乘
    total = sum(int(id_number[i]) * WEIGHT_FACTORS[i] 
                for i in range(17))
    
    # 2. 求和后对11取模
    remainder = total % 11
    
    # 3. 根据余数查表得到校验码
    return CHECK_CODES[remainder]
```

### 完整验证流程

```python
def validate(id_number: str) -> dict:
    """
    验证身份证号
    
    Returns:
        dict: {
            'valid': bool,
            'message': str,
            'info': dict  # 解析出的信息
        }
    """
    # 1. 长度检查
    if len(id_number) not in [15, 18]:
        return {'valid': False, 'message': '长度不正确'}
    
    # 2. 格式检查
    if not id_number[:17].isdigit():
        return {'valid': False, 'message': '格式不正确'}
    
    # 3. 地区码验证
    area_code = id_number[:6]
    if not is_valid_area_code(area_code):
        return {'valid': False, 'message': '地区码无效'}
    
    # 4. 出生日期验证
    birth_str = id_number[6:14]
    try:
        birth_date = datetime.strptime(birth_str, '%Y%m%d').date()
    except ValueError:
        return {'valid': False, 'message': '出生日期无效'}
    
    # 5. 校验码验证
    calculated = calculate_checksum(id_number[:17])
    if calculated != id_number[17].upper():
        return {'valid': False, 'message': '校验码不正确'}
    
    # 6. 解析信息
    info = parse_id_info(id_number)
    
    return {
        'valid': True,
        'message': '身份证号有效',
        'info': info
    }
```

### 信息解析

```python
def parse_id_info(id_number: str) -> dict:
    """解析身份证信息"""
    return {
        'area_code': id_number[:6],
        'birth_date': datetime.strptime(id_number[6:14], '%Y%m%d').date(),
        'birth_year': int(id_number[6:10]),
        'birth_month': int(id_number[10:12]),
        'birth_day': int(id_number[12:14]),
        'sequence': id_number[14:17],
        'gender': '男' if int(id_number[16]) % 2 == 1 else '女',
        'age': calculate_age(id_number[6:14])
    }
```

## 数据库操作

### 保存用户信息

```python
def save_user_info(user_data: dict):
    """保存用户信息到数据库"""
    db = Database()
    
    # 1. 检查是否已存在
    existing = db.get_user_by_id_number(user_data['id_number'])
    if existing:
        # 更新现有用户
        db.update_user(existing.id, **user_data)
    else:
        # 添加新用户
        user = db.add_user(
            name=user_data['name'],
            id_number=user_data['id_number'],
            gender=user_data['gender'],
            phone=''  # 身份证上没有电话
        )
        
        # 创建采集记录
        db.add_collection_record(
            user_id=user.id,
            status='pending'
        )
    
    # 2. 保存身份证照片
    if 'photo' in user_data:
        filepath = FileHelper.save_id_photo(
            user_data['id_number'],
            user_data['photo']
        )
        
        # 记录照片信息
        db.add_photo(
            user_id=user.id,
            filepath=filepath,
            photo_type='id_card'
        )
    
    db.close()
```

## 错误处理

### 常见错误

1. **读卡器未连接**
   ```python
   if not self.device:
       QMessageBox.warning(self, '警告', '请先连接读卡器')
       return
   ```

2. **读取失败**
   ```python
   try:
       data = self.device.read()
   except Exception as e:
       QMessageBox.critical(self, '错误', f'读取失败: {str(e)}')
       return
   ```

3. **身份证号无效**
   ```python
   result = IDCardValidator.validate(id_number)
   if not result['valid']:
       QMessageBox.warning(self, '警告', f'身份证号无效: {result["message"]}')
       return
   ```

4. **重复数据**
   ```python
   existing = db.get_user_by_id_number(id_number)
   if existing:
       reply = QMessageBox.question(
           self, '确认',
           f'用户 {existing.name} 已存在，是否更新？',
           QMessageBox.Yes | QMessageBox.No
       )
       if reply == QMessageBox.No:
           return
   ```

## 性能优化

### 1. 异步读取

```python
class ReadCardThread(QThread):
    """异步读取身份证线程"""
    finished = pyqtSignal(dict)
    error = pyqtSignal(str)
    
    def run(self):
        try:
            data = self.reader.read_card()
            self.finished.emit(data)
        except Exception as e:
            self.error.emit(str(e))
```

```python

self.recent_users = []

def add_to_cache(self, user_data):
    self.recent_users.insert(0, user_data)
    if len(self.recent_users) > 10:
        self.recent_users.pop()
```

### 3. 批量保存

```python
# 批量读取时，先缓存再批量保存
def batch_read(self, count: int):
    users = []
    for i in range(count):
        data = self.read_card()
        users.append(data)
    
    # 批量保存
    db.batch_add_users(users)
```

## 常见问题

### Q1: 读卡器无法连接？

**原因**：
- 读卡器驱动未安装
- USB接口问题
- 读卡器故障

**解决**：
1. 安装读卡器驱动
2. 更换USB接口
3. 检查读卡器是否正常工作
4. 使用测试模式

### Q2: 读取速度慢？

**原因**：
- 身份证芯片接触不良
- 读卡器性能问题

**解决**：
1. 确保身份证放置正确
2. 清洁身份证芯片
3. 更换高性能读卡器

### Q3: 照片质量差？

**原因**：
- 身份证照片本身质量问题
- 读卡器分辨率低

**解决**：
- 使用高分辨率读卡器
- 后期图像增强处理
