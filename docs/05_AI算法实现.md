## 5.1 人脸检测算法

### 5.1.1 MediaPipe Face Detection

**技术方案**: Google MediaPipe  
**模型**: BlazeFace  
**运行环境**: CPU  
**检测速度**: < 100ms/帧

**核心代码**:
```python
import mediapipe as mp

class AIProcessor:
    def __init__(self):
        self.mp_face_detection = mp.solutions.face_detection
        self.face_detection = self.mp_face_detection.FaceDetection(
            model_selection=1,  # 1=全范围模型, 0=短距离模型
            min_detection_confidence=0.7  # 最小检测置信度
        )
    
    def detect_face(self, image):
        """检测人脸"""
        rgb_image = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
        results = self.face_detection.process(rgb_image)
        
        if results.detections:
            detection = results.detections[0]
            bbox = detection.location_data.relative_bounding_box
            # 返回归一化坐标
            return {
                'x': bbox.xmin,
                'y': bbox.ymin,
                'width': bbox.width,
                'height': bbox.height,
                'confidence': detection.score[0]
            }
        return None
```

**检测流程**:
1. 图像预处理（BGR→RGB）
2. 输入MediaPipe模型
3. 获取检测结果（边界框+置信度）
4. 坐标转换（归一化→像素）

**优化策略**:
- 使用model_selection=1（全范围模型）
- 设置合理的置信度阈值（0.7）
- 跳帧检测（实时预览时每3帧检测一次）

## 5.2 AI美颜算法

### 5.2.1 GFPGAN集成

**技术方案**: GFPGAN (GAN-based Face Restoration)  
**模型版本**: 1.3  
**模型文件**: GFPGANv1.3.pth (约350MB)

**核心代码**:
```python
from gfpgan import GFPGANer

class AdvancedBeautify:
    def __init__(self):
        self.gfpgan = GFPGANer(
            model_path='gfpgan/weights/GFPGANv1.3.pth',
            upscale=1,  # 不放大
            arch='clean',
            channel_multiplier=2,
            bg_upsampler=None
        )
    
    def beautify(self, image, strength=0.8):
        """AI美颜"""
        _, _, output = self.gfpgan.enhance(
            image,
            has_aligned=False,
            only_center_face=False,
            paste_back=True,
            weight=strength  # 美颜强度 0-1
        )
        return output
```

**美颜参数**:
- `strength`: 美颜强度（0-1）
  - 0.5: 轻度美颜
  - 0.8: 中度美颜（默认）
  - 1.0: 重度美颜

**处理流程**:
1. 加载GFPGAN模型
2. 输入原始图像
3. 模型推理（人脸增强）
4. 输出美颜后图像

**效果**:
- 皮肤平滑
- 细节保留
- 自然不失真

## 5.3 背景替换算法

### 5.3.1 语义分割

**技术方案**: 基于深度学习的语义分割  
**实现方式**: 结合多种算法

**核心代码**:
```python
def replace_background(self, image, bg_color):
    """背景替换"""
    # 1. 人像分割
    mask = self.segment_person(image)
    
    # 2. 边缘优化
    mask = self.refine_edges(mask)
    
    # 3. 背景替换
    h, w = image.shape[:2]
    background = np.full((h, w, 3), bg_color, dtype=np.uint8)
    
    # 4. Alpha混合
    mask_3ch = cv2.cvtColor(mask, cv2.COLOR_GRAY2BGR) / 255.0
    result = (image * mask_3ch + background * (1 - mask_3ch)).astype(np.uint8)
    
    return result
```

**分割方法**:
1. **颜色空间分割**: HSV色彩空间阈值分割
2. **边缘检测**: Canny边缘检测
3. **形态学处理**: 开运算/闭运算优化
4. **GrabCut算法**: 精细分割

**边缘优化**:
```python
def refine_edges(self, mask):
    """边缘优化"""
    # 1. 高斯模糊
    mask = cv2.GaussianBlur(mask, (5, 5), 0)
    
    # 2. 形态学处理
    kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (5, 5))
    mask = cv2.morphologyEx(mask, cv2.MORPH_CLOSE, kernel)
    mask = cv2.morphologyEx(mask, cv2.MORPH_OPEN, kernel)
    
    # 3. 羽化边缘
    mask = cv2.GaussianBlur(mask, (3, 3), 0)
    
    return mask
```

## 5.4 智能裁剪算法

### 5.4.1 人脸定位裁剪

**算法流程**:
```
检测人脸位置
    ↓
计算人脸中心点
    ↓
根据规格计算裁剪区域
    ↓
调整位置（头顶留白）
    ↓
裁剪并缩放到目标尺寸
```

**核心代码**:
```python
def smart_crop(self, image, face_bbox, spec):
    """智能裁剪"""
    h, w = image.shape[:2]
    
    # 1. 计算人脸中心
    face_center_x = face_bbox['x'] + face_bbox['width'] / 2
    face_center_y = face_bbox['y'] + face_bbox['height'] / 2
    
    # 2. 计算裁剪区域
    crop_width, crop_height = spec['size']  # 目标尺寸
    aspect_ratio = crop_width / crop_height
    
    # 3. 确定裁剪框
    if w / h > aspect_ratio:
        # 图像更宽，以高度为准
        crop_h = h
        crop_w = int(h * aspect_ratio)
    else:
        # 图像更高，以宽度为准
        crop_w = w
        crop_h = int(w / aspect_ratio)
    
    # 4. 调整位置（人脸居中，头顶留白）
    offset_y = int(face_bbox['height'] * 0.3)  # 头顶留30%人脸高度
    center_x = int(face_center_x * w)
    center_y = int(face_center_y * h) - offset_y
    
    x1 = max(0, center_x - crop_w // 2)
    y1 = max(0, center_y - crop_h // 2)
    x2 = min(w, x1 + crop_w)
    y2 = min(h, y1 + crop_h)
    
    # 5. 裁剪
    cropped = image[y1:y2, x1:x2]
    
    # 6. 缩放到目标尺寸
    result = cv2.resize(cropped, (crop_width, crop_height))
    
    return result
```

## 5.5 质量评分算法

### 5.5.1 多维度评分

**评分维度**:
1. **清晰度** (30%)
2. **光照均匀度** (25%)
3. **人脸位置** (25%)
4. **表情自然度** (20%)

**核心代码**:
```python
def calculate_quality_score(self, image, face_bbox):
    """计算照片质量分数"""
    scores = {}
    
    # 1. 清晰度评分 (Laplacian方差)
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    laplacian_var = cv2.Laplacian(gray, cv2.CV_64F).var()
    scores['sharpness'] = min(100, laplacian_var / 10)
    
    # 2. 光照评分 (亮度标准差)
    brightness_std = np.std(gray)
    scores['lighting'] = max(0, 100 - abs(brightness_std - 50))
    
    # 3. 人脸位置评分
    center_x = face_bbox['x'] + face_bbox['width'] / 2
    center_y = face_bbox['y'] + face_bbox['height'] / 2
    position_score = 100 - abs(center_x - 0.5) * 100 - abs(center_y - 0.4) * 100
    scores['position'] = max(0, position_score)
    
    # 4. 人脸大小评分
    face_area = face_bbox['width'] * face_bbox['height']
    size_score = min(100, face_area * 400)
    scores['size'] = size_score
    
    # 5. 综合评分
    total_score = (
        scores['sharpness'] * 0.30 +
        scores['lighting'] * 0.25 +
        scores['position'] * 0.25 +
        scores['size'] * 0.20
    )
    
    return {
        'total': round(total_score, 1),
        'details': scores
    }
```

**评分标准**:
- 90-100分: 优秀
- 80-89分: 良好
- 70-79分: 合格
- < 70分: 需重拍

## 5.6 算法性能优化

### 5.6.1 模型加载优化
```python
# 单例模式，避免重复加载
class AIProcessor:
    _instance = None
    
    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
            cls._instance._initialized = False
        return cls._instance
    
    def __init__(self):
        if self._initialized:
            return
        # 加载模型
        self.load_models()
        self._initialized = True
```
