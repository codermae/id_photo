# 用户手册转换说明

## 如何将Markdown用户手册转换为Word文档

### 方法1：使用pandoc（推荐）

1. **安装pandoc**
```bash
# Windows: 下载安装包 https://github.com/jgm/pandoc/releases
# 或使用包管理器
choco install pandoc  # Chocolatey
scoop install pandoc  # Scoop
```

2. **转换命令**
```bash
cd id_photo_system/docs
pandoc 16_智能化证件照采集及处理系统用户手册.md -o 智能化证件照采集及处理系统用户手册.docx
```

### 方法2：使用Python的python-docx库

1. **安装依赖**
```bash
pip install python-docx markdown
```

2. **创建转换脚本**
```python
# convert_to_word.py
from docx import Document
from docx.shared import Pt, Inches
from docx.enum.text import WD_ALIGN_PARAGRAPH
import markdown

def markdown_to_word(md_file, docx_file):
    # 读取Markdown文件
    with open(md_file, 'r', encoding='utf-8') as f:
        md_content = f.read()
    
    # 创建Word文档
    doc = Document()
    
    # 设置文档属性
    doc.core_properties.title = "智能化证件照采集及处理系统用户手册"
    doc.core_properties.author = "系统开发团队"
    doc.core_properties.subject = "用户手册"
    
    # 添加标题
    title = doc.add_heading('智能化证件照采集及处理系统用户手册', 0)
    title.alignment = WD_ALIGN_PARAGRAPH.CENTER
    
    # 添加版本信息
    version = doc.add_paragraph('版本: 1.0')
    version.alignment = WD_ALIGN_PARAGRAPH.CENTER
    
    release_date = doc.add_paragraph('发布日期: 2026年5月')
    release_date.alignment = WD_ALIGN_PARAGRAPH.CENTER
    
    audience = doc.add_paragraph('适用对象: 系统管理员、操作员、最终用户')
    audience.alignment = WD_ALIGN_PARAGRAPH.CENTER
    
    # 添加目录
    doc.add_page_break()
    toc = doc.add_heading('目录', 1)
    
    # 这里可以继续添加Markdown内容的转换逻辑
    # 由于Markdown到Word的完整转换较复杂，建议使用pandoc
    
    # 保存文档
    doc.save(docx_file)
    print(f"已保存到: {docx_file}")

if __name__ == "__main__":
    markdown_to_word('16_智能化证件照采集及处理系统用户手册.md', 
                     '智能化证件照采集及处理系统用户手册.docx')
```

### 推荐的Word文档格式设置

#### 页面设置
- 页边距：上下2.54cm，左右3.17cm
- 纸张大小：A4
- 页眉：系统名称和版本
- 页脚：页码和日期

#### 样式设置
1. **标题1**：黑体，18pt，加粗，居中
2. **标题2**：黑体，16pt，加粗，左对齐
3. **标题3**：宋体，14pt，加粗，左对齐
4. **正文**：宋体，12pt，1.5倍行距
5. **代码**：Consolas，11pt，灰色背景

#### 封面设计
1. 系统名称：大号字体，居中
2. 版本信息：居中
3. 公司Logo：右上角
4. 文档编号和日期：底部

### 批量处理脚本示例

```python
# batch_convert.py - 批量转换所有文档
import os
import subprocess

def convert_all_md_to_docx(docs_dir):
    """转换docs目录下所有Markdown文件为Word"""
    for filename in os.listdir(docs_dir):
        if filename.endswith('.md') and not filename.startswith('_'):
            md_path = os.path.join(docs_dir, filename)
            docx_path = os.path.join(docs_dir, filename.replace('.md', '.docx'))
            
            # 使用pandoc转换
            cmd = ['pandoc', md_path, '-o', docx_path]
            try:
                subprocess.run(cmd, check=True)
                print(f"✓ 已转换: {filename} → {filename.replace('.md', '.docx')}")
            except Exception as e:
                print(f"✗ 转换失败: {filename} - {e}")

if __name__ == "__main__":
    convert_all_md_to_docx('id_photo_system/docs')
```

### 注意事项

1. **编码问题**
   - 确保使用UTF-8编码
   - 中文字符可能需要在Word中调整字体

2. **图片路径**
   - Markdown中的图片链接需要转换为绝对路径
   - 或使用Word插入图片功能

3. **复杂格式**
   - 复杂表格可能需要手动调整
   - 代码块可能需要特殊处理

4. **版本控制**
   - 保留Markdown源文件
   - 每次更新先更新Markdown文件
   - 再重新生成Word文档

### 快速转换命令（使用pandoc）

```bash
# 基本转换
pandoc 16_智能化证件照采集及处理系统用户手册.md -o 用户手册.docx

# 带样式的转换
pandoc 16_智能化证件照采集及处理系统用户手册.md \
  --reference-doc=custom-reference.docx \
  -o 用户手册.docx

# 批量转换
for file in *.md; do
  pandoc "$file" -o "${file%.md}.docx"
done
```
