# 数据库快速参考卡

## 表结构速查

### Collections (采集任务)
```
id (PK)          | 主键
name             | 任务名称
organization     | 机构名称
status           | 状态: active/completed/archived
start_date       | 开始日期
end_date         | 结束日期
created_at       | 创建时间
updated_at       | 更新时间
```

### Users (用户信息)
```
id (PK)          | 主键
collection_id    | 采集任务ID (FK)
name             | 姓名
id_number        | 身份证号
gender           | 性别
nation           | 民族
birthday         | 出生日期
address          | 地址
created_at       | 创建时间
updated_at       | 更新时间
```

### Photos (照片记录)
```
id (PK)          | 主键
user_id          | 用户ID (FK)
photo_type       | 类型: raw/processed
file_path        | 文件路径
file_size        | 文件大小
photo_spec       | 规格: 一寸/二寸等
background_color | 背景色: white/blue/red等
quality_score    | 质量评分 (0-100)
face_similarity  | 相似度 (0-100)
created_at       | 创建时间
```

### CollectionRecords (采集记录)
```
id (PK)          | 主键
collection_id    | 采集任务ID (FK)
user_id          | 用户ID (FK)
collection_date  | 采集日期
operator         | 操作员
status           | 状态: pending/processing/completed
notes            | 备注
created_at       | 创建时间
```

## 常用查询

### 1. 查询采集进度
```python
# 查询各状态的用户数
session.query(CollectionRecord.status, func.count(CollectionRecord.id))\
    .filter(CollectionRecord.collection_id == collection_id)\
    .group_by(CollectionRecord.status)\
    .all()
```

### 2. 查询用户及其照片
```python
# 查询用户及其所有照片
session.query(User, Photo)\
    .outerjoin(Photo)\
    .filter(User.collection_id == collection_id)\
    .all()
```

### 3. 查询采集完成率
```python
# 计算完成率
completed = session.query(CollectionRecord)\
    .filter(CollectionRecord.collection_id == collection_id,
            CollectionRecord.status == 'completed')\
    .count()
total = session.query(User)\
    .filter(User.collection_id == collection_id)\
    .count()
rate = completed / total * 100 if total > 0 else 0
```

### 4. 按性别统计
```python
# 按性别统计用户数
session.query(User.gender, func.count(User.id))\
    .filter(User.collection_id == collection_id)\
    .group_by(User.gender)\
    .all()
```

### 5. 按日期统计采集进度
```python
# 按日期统计采集记录
session.query(CollectionRecord.collection_date, 
              CollectionRecord.status,
              func.count(CollectionRecord.id))\
    .filter(CollectionRecord.collection_id == collection_id)\
    .group_by(CollectionRecord.collection_date, CollectionRecord.status)\
    .order_by(CollectionRecord.collection_date)\
    .all()
```

## 状态值速查

### 采集任务状态 (collections.status)
- `active` - 进行中
- `completed` - 已完成
- `archived` - 已归档

### 照片类型 (photos.photo_type)
- `raw` - 原始照片
- `processed` - 处理后照片

### 采集状态 (collection_records.status)
- `pending` - 待采集
- `processing` - 待处理
- `completed` - 已完成

## 数据流向速查

```
创建任务 → 导入用户 → 拍照 → 处理 → 统计

collections ← 1条记录
users ← n条记录
collection_records ← n条记录 (status='pending')
photos ← n条原始照片 (photo_type='raw')
collection_records ← 更新为 'processing'
photos ← n*m条处理后照片 (photo_type='processed')
collection_records ← 更新为 'completed'
```

## 常见问题

### Q: 如何查询一个用户的所有照片？
```python
user = session.query(User).get(user_id)
photos = user.photos  # 直接访问关系
```

### Q: 如何查询一个采集任务的所有用户？
```python
collection = session.query(Collection).get(collection_id)
users = collection.users  # 直接访问关系
```

### Q: 如何查询一个用户的采集记录？
```python
user = session.query(User).get(user_id)
records = user.records  # 直接访问关系
```

### Q: 如何删除一个采集任务及其所有数据？
```python
collection = session.query(Collection).get(collection_id)
session.delete(collection)  # 级联删除所有相关数据
session.commit()
```

### Q: 如何统计一个采集任务的完成率？
```python
completed = session.query(CollectionRecord)\
    .filter(CollectionRecord.collection_id == collection_id,
            CollectionRecord.status == 'completed')\
    .count()
total = session.query(User)\
    .filter(User.collection_id == collection_id)\
    .count()
rate = (completed / total * 100) if total > 0 else 0
```

## 性能提示

1. **使用索引字段进行查询**
   - 优先使用 collection_id, user_id, id_number 等索引字段
   - 避免全表扫描

2. **使用分页查询**
   - 大数据量时使用 limit 和 offset
   - 避免一次性加载所有数据

3. **使用批量操作**
   - 批量插入而不是逐条插入
   - 使用事务确保数据一致性

4. **缓存热数据**
   - 缓存采集任务列表
   - 缓存统计结果
