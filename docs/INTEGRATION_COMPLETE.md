## 🔧 技术实现详情

### 1. 规格去重算法
```python
# 按尺寸分组去重
size_groups = defaultdict(list)
for spec_name, size in PHOTO_SPECS.items():
    size_groups[size].append(spec_name)

# 界面显示优化
for size, specs in size_groups.items():
    if len(specs) == 1:
        display_name = specs[0]
    else:
        display_name = f"{specs[0]} 等{len(specs)}种"
        checkbox.specs_group = specs  # 存储完整规格组
```

# 新增批量模式选择
self.batch_mode_combo = QComboBox()

## 🚀 使用指南

### 1. 正常图像处理
1. 加载图像后，在"证件照规格"下拉框中选择规格
2. 界面显示优化后的10种不同尺寸选项
3. 鼠标悬停可查看包含的具体规格详情
4. 点击"📷 与原图对比"可对比处理效果

### 2. 多规格同时生成
1. 加载图像后，点击"🎯 多规格同时生成"按钮
2. 在弹出对话框中选择需要的规格和颜色
3. 使用快速选择按钮快速配置常用组合
4. 确认后自动生成所有组合的照片

### 3. 批量处理
1. 点击"批量处理"按钮打开批量处理对话框
2. 选择"批量模式"：单规格处理 或 多规格生成
3. **单规格模式**: 每张图片生成一种规格
4. **多规格模式**: 每张图片生成多种规格组合
5. 配置处理参数后开始批量处理
