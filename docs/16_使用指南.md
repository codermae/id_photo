## 快速开始

### 安装和启动

```bash
# 1. 安装依赖
cd id_photo_system
pip install -r requirements.txt

# 2. 启动程序
python main.py
```

### 完整流程

```
添加用户 → 拍照采集 → 图像处理 → 数据管理
```

#### 1. 添加用户

**方式A: 身份证读取**（需要读卡器）
- 切换到"身份证读取"标签
- 点击"放身份证"（测试模式自动填充）
- 系统自动保存

**方式B: 手动录入**
- 切换到"数据导入"标签
- 填写姓名、身份证号等信息
- 点击"添加用户"

**方式C: Excel批量导入**
- 准备Excel文件（姓名、身份证号、性别、电话）
- 点击"导入Excel"
- 选择文件，系统自动导入

#### 2. 拍照采集

- 切换到"拍照采集"标签
- 点击"打开摄像头"
- 等待质量提示变绿（人脸检测、光线充足）
- 点击"拍照"
- 照片自动保存到 `data/photos/raw/`

#### 3. 图像处理

- 切换到"图像处理"标签
- 点击"选择图像"，选择照片
- 系统自动识别用户
- 选择规格（一寸、二寸等）
- 选择背景色（白色、蓝色、红色）
- 调整美颜强度（1-5级）
- 点击"处理"
- 点击"保存"（自动保存到 `data/photos/processed/`）

#### 4. 数据管理

- 切换到"数据管理"标签
- 查看所有用户和采集状态
- 搜索用户（姓名或身份证号）
- 筛选状态（全部/待采集/已采集）
- 批量删除（Ctrl/Shift多选）

## 功能详解

### 1. 身份证读取

**功能**：读取身份证信息并自动保存

**操作步骤**：
1. 连接身份证读卡器
2. 点击"连接读卡器"
3. 放置身份证
4. 点击"读取身份证"
5. 系统自动保存用户信息和照片

**测试模式**：
- 无读卡器时使用
- 点击"放身份证"自动填充测试数据
- 用于演示和测试

**自动功能**：
- ✅ 自动创建待采集记录
- ✅ 自动保存身份证照片
- ✅ 自动更新状态栏

---

### 2. 数据导入

**功能**：手动添加或批量导入用户

**手动添加**：
1. 填写用户信息
   - 姓名（必填）
   - 身份证号（必填，自动校验）
   - 性别（自动识别）
   - 电话（可选）
2. 点击"添加用户"

**批量导入**：
1. 准备Excel文件，包含列：
   - 姓名
   - 身份证号
   - 性别
   - 电话
2. 点击"导入Excel"
3. 选择文件
4. 系统自动导入并验证

**身份证校验**：
- 符合GB 11643-1999标准
- 自动验证校验码
- 自动识别性别和年龄
- 实时提示错误

**自动功能**：
- ✅ 自动创建待采集记录
- ✅ 自动更新状态栏

---

### 3. 拍照采集

**功能**：使用摄像头拍摄证件照

**操作步骤**：
1. 选择摄像头（默认0）
2. 点击"打开摄像头"
3. 调整拍摄位置
4. 观察质量提示：
   - 🟢 绿色：质量良好，可以拍照
   - 🟡 黄色：需要调整
   - 🔴 红色：质量不佳
5. 点击"拍照"
6. 照片自动保存

**质量检测**：
- ✅ 人脸检测（必须检测到人脸）
- ✅ 光线检测（亮度适中）
- ✅ 清晰度检测（无模糊）
- ✅ 位置检测（人脸居中）

**自动功能**：
- ✅ 自动保存到 `data/photos/raw/`
- ✅ 自动更新采集记录为"已采集"
- ✅ 自动更新状态栏

**快捷键**：
- 空格键：拍照
- Esc：关闭摄像头

**美颜效果**：

**自动功能**：
- ✅ 自动识别用户

- ✅ 文件名格式：`{身份证号}_{规格}_{背景}_{时间}.jpg`

## 批量处理

### 批量导入用户

**准备Excel文件**：

| 姓名 | 身份证号 | 性别 | 电话 |
|------|----------|------|------|
| 张三 | 110101199001010015 | 男 | 13800138000 |
| 李四 | 110101199001010023 | 女 | 13800138001 |

**导入步骤**：
1. 切换到"数据导入"标签
2. 点击"导入Excel"
3. 选择文件
4. 系统自动导入并验证

### 批量处理照片

使用批量处理脚本：

```bash
python -c "
from controllers.batch_processor import BatchProcessor

processor = BatchProcessor()
results = processor.process_batch(
    input_dir='data/photos/raw/',
    output_dir='data/photos/processed/',
    spec='一寸',
    bg_color='白色',
    beautify_level=3
)

for result in results:
    print(f'{result[\"filename\"]}: {result[\"status\"]}')
"
```

**参数说明**：
- `input_dir`: 输入目录
- `output_dir`: 输出目录
- `spec`: 照片规格
- `bg_color`: 背景颜色
- `beautify_level`: 美颜强度（1-5）

## 常见问题

### Q1: 身份证号提示"校验码不正确"？

**原因**：身份证号不符合GB 11643-1999标准

**解决**：
1. 检查身份证号是否输入正确
2. 使用 `generate_valid_id.py` 生成测试数据
3. 确认最后一位校验码正确

### Q2: 摄像头打不开？

**原因**：摄像头被占用或驱动问题

**解决**：
1. 关闭其他使用摄像头的程序
2. 尝试更换摄像头ID（0, 1, 2...）
3. 检查摄像头驱动

### Q3: 保存图片提示"文件不存在"？

**原因**：中文路径问题（已修复）

**解决**：
- 系统已使用 `cv2.imencode()` 支持中文路径
- 如仍有问题，检查文件权限

### Q4: 背景替换效果不好？

**原因**：照片质量或光线问题

**解决**：
1. 确保光线充足均匀
2. 背景尽量单色
3. 人脸清晰无遮挡
4. 使用手动裁剪微调

### Q5: 数据管理不显示最新数据？

**原因**：已修复，现在自动刷新

**功能**：
- ✅ 切换标签页自动刷新
- ✅ 数据变更自动通知
- ✅ 状态栏实时更新

### Q6: 如何调整美颜参数？

**方法1**：界面调整
- 在"图像处理"中调整美颜强度（1-5级）

**方法2**：参数调试工具
```bash
python parameter_tuning_debug.py
```
- 实时预览效果
- 调整详细参数
- 保存最佳配置

### Q7: 如何备份数据？

**备份内容**：
```
data/
├── database/        # 数据库
├── photos/          # 所有照片
└── exports/         # 导出文件
```

**备份方法**：
- 直接复制整个 `data/` 目录
- 定期备份到其他位置

### Q8: 支持哪些照片规格？

**当前支持**：
- 一寸（25×35mm）
- 小二寸（35×49mm）
- 二寸（35×53mm）
- 大一寸（33×48mm）

**添加新规格**：
在 `config/config.py` 中添加：
```python
PHOTO_SPECS['护照'] = (354, 472)
```

### Q9: 如何生成测试数据？

```bash
# 生成有效的测试身份证号
python generate_valid_id.py

# 初始化采集记录
python init_collection_records.py
```

### Q10: 系统运行慢怎么办？

**优化建议**：
1. 使用GPU加速（安装CUDA）
2. 减少美颜强度
3. 关闭不必要的程序
4. 定期清理临时文件
