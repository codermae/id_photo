## 🔧 技术实现

### 1. 规格分组算法
```python
# 按尺寸分组
size_groups = defaultdict(list)
for spec_name in PHOTO_SPECS.keys():
    size = PHOTO_SPECS[spec_name]
    size_groups[size].append(spec_name)

# 每组选择代表
for size, specs in size_groups.items():
    representative = specs[0]  # 选择第一个作为代表
```

### 2. 界面适配
```python
if len(specs) == 1:
    checkbox = QCheckBox(specs[0])
else:
    checkbox = QCheckBox(f"{specs[0]} 等{len(specs)}种")
    checkbox.specs_group = specs
```

### 3. 生成逻辑
```python
for checkbox in selected_checkboxes:
    if hasattr(checkbox, 'specs_group'):
        # 多规格组，只生成一张
        spec = checkbox.specs_group[0]
    else:
        # 单一规格
        spec = checkbox.text()
    generate_photo(spec, color)
```
