# 新功能使用指南

## 🎉 功能概览

系统已成功添加四项重要功能，大幅提升了证件照处理的效率和用户体验：

### ✅ 已完成功能
1. **统计报表按采集任务分类导出** - 精细化数据管理
2. **多规格同时生成** - 批量处理效率提升
3. **原图对比预览** - 直观效果查看
4. **全球主流证件照规格扩充** - 国际化支持

## 📊 1. 统计报表分类导出

### 使用步骤
1. 打开"统计报表"标签页
2. 选择采集任务（或选择"全部采集任务"）
3. 设置统计日期范围
4. 点击"导出报表"按钮
5. 选择保存位置

### 导出内容
- **单个任务**: 统计汇总 + 用户详细列表
- **全部任务**: 任务对比汇总 + 各任务用户明细（多工作表）

### 数据字段
- 采集任务信息、组织机构、完成率
- 用户详细信息、照片统计、采集状态
- 时间统计、操作员记录

## 🎯 2. 多规格同时生成

### 使用步骤
1. 在"图像处理"页面加载图像
2. 点击"🎯 多规格同时生成"按钮
3. 选择需要的证件照规格（支持多选）
4. 选择需要的背景颜色（支持多选）
5. 设置处理选项（美颜、处理模式）
6. 点击"🚀 开始生成"

### 快速选择
- **全选**: 选择所有规格
- **全不选**: 清除所有选择
- **选择常用**: 一寸、小二寸、二寸等中国常用规格
- **选择国际**: 美国护照、欧盟护照等国际规格

### 生成结果
- 自动生成 规格数×颜色数 张照片
- 智能文件命名：`身份证号_规格_背景色_时间戳.jpg`
- 实时进度显示和完成统计

## 📷 3. 原图对比预览

### 使用步骤
1. 加载图像后，系统自动保存原图
2. 进行任意图像处理操作
3. 点击"📷 与原图对比"查看效果
4. 再次点击可关闭对比模式

### 显示模式
- **普通模式**: 单一预览窗口显示处理后图像
- **对比模式**: 左侧处理后，右侧原图，并排显示

### 状态提示
- 实时显示当前预览模式
- 清晰标识"处理后"和"原图"

## 🚀 使用建议

### 多规格生成最佳实践
1. **合理选择**: 根据实际需求选择规格，避免生成过多文件
2. **批量大小**: 建议单次不超过50张（10规格×5颜色）
3. **处理模式**: 推荐"高保真模式"获得最佳效果
4. **背景选择**: 优先选择目标用途的标准背景色

### 原图对比技巧
1. **处理前查看**: 先了解原图质量和特点
2. **实时对比**: 每次处理后都对比查看效果
3. **细节检查**: 重点关注人脸、头发、边缘区域
4. **参数调整**: 根据对比结果调整处理参数

### 报表导出建议
1. **定期导出**: 建议每周或每月定期导出
2. **分类管理**: 大型组织按采集任务分别导出
3. **数据备份**: 导出报表可作为重要数据备份
4. **深度分析**: 利用Excel功能进行数据分析

## ⚠️ 注意事项

### 系统要求
- Python 3.8+ 环境
- 足够的磁盘空间（多规格生成会产生大量文件）
- 建议内存 4GB+ （处理大批量时）

### 可选依赖
- `face_recognition`: 人脸识别功能（可选）
- `face-detection-tflite`: 高精度人脸检测（可选）
- 系统会自动降级到备选方案，不影响核心功能

### 文件管理
- 多规格生成的文件会自动保存到 `data/photos/processed/` 目录
- 文件名包含完整信息，便于识别和管理
- 建议定期清理不需要的文件

## 🔧 故障排除

### 常见问题

**Q: 多规格生成失败**
A: 检查磁盘空间，确保有足够存储空间

**Q: 原图对比不显示**
A: 确保已加载图像，系统会自动保存原图

**Q: 报表导出为空**
A: 检查选择的日期范围和采集任务是否有数据

**Q: 新规格不显示**
A: 重启程序，确保配置文件已更新

### 性能优化
- 大批量生成时建议关闭其他程序
- 使用SSD硬盘可提升处理速度
- 定期清理临时文件和缓存
