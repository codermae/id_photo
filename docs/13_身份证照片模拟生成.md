# 身份证照片模拟生成

## 功能概述

身份证照片模拟生成模块提供了一套完整的工具，用于生成逼真的模拟身份证照片。这使得系统在没有真实读卡器的情况下，也能完整地模拟身份证读取过程，包括返回身份证照片。

---

## 核心特性

### 1. 完整的身份证照片生成

- 生成标准的身份证照片（85×54mm）
- 包含个人照片区域
- 包含文字信息（姓名、身份证号等）
- 包含防伪元素（国徽、文字等）

### 2. 灵活的生成方式

- **单张生成**: 为单个用户生成身份证照片
- **批量生成**: 为多个用户批量生成
- **集成读卡器**: 与读卡器模拟器无缝集成

### 3. 高度可定制

- 支持自定义用户信息
- 支持自定义背景颜色
- 支持自定义字体
- 支持添加真实人脸照片

## 使用方法

### 方法1: 直接使用模拟器

```python
from utils.id_card_photo_generator import IDCardPhotoSimulator
import cv2

# 创建模拟器
simulator = IDCardPhotoSimulator()

# 用户信息
user_data = {
    'name': '张三',
    'id_number': '110101199001010015',
    'gender': '男',
    'birth_date': '1990-01-01',
    'nation': '汉',
    'address': '北京市东城区XX街道XX号',
    'issue_authority': '北京市公安局东城分局',
    'valid_period': '2010.01.01-2030.01.01'
}

# 生成身份证照片
id_card = simulator.generate_realistic_id_card(user_data)

# 保存照片
cv2.imwrite('id_card.jpg', id_card)

# 显示照片
cv2.imshow('ID Card', id_card)
cv2.waitKey(0)
```

### 方法2: 批量生成

```python
# 用户列表
users = [
    {'name': '张三', 'id_number': '110101199001010015', ...},
    {'name': '李四', 'id_number': '110101199001010023', ...},
    {'name': '王五', 'id_number': '310101198503151232', ...}
]

# 批量生成
id_cards = simulator.generate_batch_id_cards(users)

# 保存所有照片
for user, id_card in zip(users, id_cards):
    filename = f"id_card_{user['id_number']}.jpg"
    cv2.imwrite(filename, id_card)
```

### 工作流程

```

   ↓
2. 调用 reader.insert_test_card(...)
   ↓
3. 读卡器创建用户信息字典
```

## 常见问题

### Q1: 生成的照片太小？

**解决**:
```python
# 调整输出尺寸
id_card = cv2.resize(id_card, (800, 500))
```

### Q2: 文字显示不正确？

**原因**: 字体路径不正确或不支持中文

**解决**:
```python
# 检查字体路径
import os
font_path = 'C:\\Windows\\Fonts\\simhei.ttf'
if os.path.exists(font_path):
    generator.font_path = font_path
```

### Q3: 如何添加真实人脸照片？

**方法**:

```python
# 读取人脸照片
face_photo = cv2.imread('face.jpg')

id_card = simulator.generate_realistic_id_card(user_data, face_photo)
```
