## 🔧 技术架构

### MVC架构
```
Model层（数据模型）
  ├─ User（用户）
  ├─ Photo（照片）
  ├─ Collection（采集任务）
  └─ CollectionRecord（采集记录）

View层（用户界面）
  ├─ MainWindow（主窗口）
  ├─ CameraView（拍照）
  ├─ DataView（数据管理）
  ├─ ReportView（统计报表）
  └─ 其他视图

Controller层（业务逻辑）
  ├─ AIProcessor（AI处理）
  ├─ ImageProcessor（图像处理）
  ├─ BatchProcessor（批量处理）
  ├─ IDCardReader（身份证读取）
  ├─ StatisticsAnalyzer（统计分析）
  └─ 其他控制器
```

### 技术栈
```
语言：Python 3.8+
GUI：PyQt5 5.15+
数据库：SQLite 3.x + SQLAlchemy 1.4+
AI库：TensorFlow 2.16+, PyTorch
图像：OpenCV 4.5+, Pillow
数据：pandas, numpy, scikit-image
可视化：ECharts 5.x
工具：日志、缓存、错误处理
```
