## 安装步骤

### 1. 安装Python

#### Windows

```bash
# 下载Python 3.12
# https://www.python.org/downloads/

# 安装时勾选"Add Python to PATH"
# 验证安装
python --version
pip --version
```

#### macOS

```bash
# 使用Homebrew安装
brew install python@3.12

# 验证安装
python3 --version
pip3 --version
```

#### Linux (Ubuntu/Debian)

```bash
# 安装Python和pip
sudo apt-get update
sudo apt-get install python3.12 python3-pip

# 验证安装
python3 --version
pip3 --version
```

### 2. 克隆项目

```bash
# 克隆项目
git clone <repository-url>
cd id_photo_system

# 查看项目结构
ls -la
```

### 3. 创建虚拟环境

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate

# 验证激活
which python  # 应该显示虚拟环境路径
```

### 4. 安装依赖

```bash
# 升级pip
pip install --upgrade pip

# 安装依赖
pip install -r requirements.txt

# 验证安装
pip list
```

**Windows 用户注意**：如果 dlib 安装失败，使用国内镜像：

```bash
# 使用清华镜像安装 dlib
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple dlib>=19.24.0

# 使用清华镜像安装 face-recognition
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple face-recognition>=1.3.0
```

### 5. 初始化数据库

```bash
# 运行初始化脚本
python fix_database_and_setup.py

# 或手动初始化
python -c "from config.database import init_db; init_db()"
```

### 6. 启动应用

```bash
# 运行主程序
python main.py
```

## 常见问题解决

### 问题1: dlib安装失败

**症状**：`error: Microsoft Visual C++ 14.0 or greater is required`

**解决方案**：

```bash
# 方案1：安装Visual C++构建工具
# 下载：https://visualstudio.microsoft.com/visual-cpp-build-tools/

# 方案2：使用预编译wheel
pip install dlib --only-binary :all:

# 方案3：使用conda
conda install -c conda-forge dlib
```

### 问题2: TensorFlow安装失败

**症状**：`No module named 'tensorflow'`

**解决方案**：

```bash
# 检查Python版本（需要3.9-3.12）
python --version

# 升级pip
pip install --upgrade pip

# 重新安装TensorFlow
pip install tensorflow>=2.16.0

# 验证安装
python -c "import tensorflow as tf; print(tf.__version__)"
```

### 问题3: OpenCV导入错误

**症状**：`ImportError: libGL.so.1`

**解决方案**：

```bash
# Linux系统
sudo apt-get install libgl1-mesa-glx

# 或使用headless版本
pip install opencv-python-headless
```

### 问题4: 摄像头无法识别

**症状**：`cv2.error: (-215:Assertion failed)`

**解决方案**：

```bash
# 检查摄像头权限
# Windows：检查设备管理器
# macOS：系统偏好设置 → 安全性与隐私
# Linux：sudo usermod -a -G video $USER

# 测试摄像头
python -c "import cv2; cap = cv2.VideoCapture(0); print(cap.isOpened())"
```

### 问题5: 人脸识别库导入失败

**症状**：`ImportError: cannot import name 'face_recognition'`

**解决方案**：

```bash
# 确保dlib已安装
pip install dlib>=19.24.0

# 安装face-recognition
pip install face-recognition>=1.3.0

# 验证安装
python -c "import face_recognition; print(face_recognition.__version__)"
```

