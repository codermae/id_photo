# 系统架构设计

## 架构概述

系统采用经典的MVC (Model-View-Controller) 架构，分为表示层、业务逻辑层、数据访问层和工具层四个层次。这种分层设计提高了代码的可维护性、可扩展性和可测试性。

**架构特点**：
- 清晰的分层结构
- 低耦合高内聚
- 易于扩展和维护
- 支持多线程处理
- 完整的错误处理

---

## 架构分层

### 1. 表示层 (Views)

**职责**：用户界面展示和交互

```
views/
├── main_window.py              # 主窗口
├── camera_view.py              # 摄像头视图
├── process_view.py             # 图像处理视图
├── data_view.py                # 数据管理视图
├── report_view.py              # 统计报表视图
├── id_card_view.py             # 身份证读取视图
├── import_view.py              # 数据导入视图
├── batch_processing_dialog.py  # 批量处理对话框
└── crop_dialog.py              # 裁剪对话框
```

**主要功能**：
- 显示用户界面
- 处理用户输入
- 调用业务逻辑层
- 显示处理结果

**技术栈**：
- PyQt5：GUI框架
- Qt Designer：界面设计

### 2. 业务逻辑层 (Controllers)

**职责**：处理业务逻辑和数据处理

```
controllers/
├── image_processor.py          # 图像处理控制器
├── smart_cropper.py            # 智能裁剪器
├── background_replacer.py      # 背景替换器
├── realtime_adjuster.py        # 实时调整器
├── advanced_beautify.py        # 高级美颜处理
├── ai_processor.py             # AI处理控制器
├── camera.py                   # 摄像头控制
├── batch_processor.py          # 批量处理
├── id_card_reader.py           # 身份证读取
├── duplicate_checker.py        # 重复检测
└── identity_verifier.py        # 身份验证
```

**主要功能**：
- 图像处理和转换
- AI算法调用
- 数据验证和转换
- 业务流程控制

**技术栈**：
- OpenCV：图像处理
- rembg：背景分割
- face-recognition：人脸识别
- MediaPipe：人脸检测

### 3. 数据访问层 (Models)

**职责**：数据模型定义和数据库操作

```
models/
├── user.py                     # 用户模型
├── photo.py                    # 照片模型
└── record.py                   # 采集记录模型
```

**主要功能**：
- 定义数据模型
- 数据库表映射
- 数据关系管理

**技术栈**：
- SQLAlchemy：ORM框架
- SQLite：数据库

### 4. 工具层 (Utils)

**职责**：提供通用工具函数

```
utils/
├── image_helper.py             # 图像处理助手
├── database_helper.py          # 数据库操作助手
├── id_card_validator.py        # 身份证校验
├── file_helper.py              # 文件处理
├── face_recognition_engine.py  # 人脸识别引擎
├── face_recognition_manager.py # 人脸识别管理器
└── voice_helper.py             # 语音助手
```

**主要功能**：
- 图像处理辅助函数
- 数据库操作辅助
- 文件处理
- 数据验证

## 系统模块

### 核心模块

#### 1. 身份证读取模块
```
身份证读取 → 信息提取 → 数据验证 → 照片生成 → 数据存储
```

#### 2. 数据导入模块
```
Excel导入 → 数据解析 → 数据验证 → 批量添加 → 数据存储
```

#### 3. 拍照采集模块
```
摄像头启动 → 实时预览 → 质量检测 → 照片保存 → 数据记录
```

#### 4. 图像处理模块
```
原始照片 → 智能裁剪 → 背景替换 → 美颜处理 → 参数调整 → 最终照片
```

#### 5. 数据管理模块
```
用户管理 → 照片管理 → 采集记录 → 数据统计 → 数据导出
```

#### 6. 人脸识别模块
```
人脸检测 → 特征提取 → 特征比对 → 防重复采集 → 身份核验
```

## 数据流

### 用户采集流程

```
┌─────────────────────────────────────────────────────────┐
│                    用户采集流程                          │
└─────────────────────────────────────────────────────────┘

1. 身份证读取
   ├─ 读卡器读取 / 测试模式
   ├─ 信息提取和验证
   └─ 生成身份证照片

2. 数据导入
   ├─ 手动添加 / Excel导入
   ├─ 数据验证
   └─ 保存到数据库

3. 拍照采集
   ├─ 摄像头启动
   ├─ 实时质量检测
   ├─ 人脸识别防重复
   └─ 保存原始照片

4. 图像处理
   ├─ 智能裁剪
   ├─ 背景替换
   ├─ 美颜处理
   └─ 保存处理后照片

5. 数据管理
   ├─ 查询和统计
   ├─ 数据导出
   └─ 报表生成
```

### 人脸识别流程

```
┌─────────────────────────────────────────────────────────┐
│                  人脸识别流程                            │
└─────────────────────────────────────────────────────────┘

1. 人脸检测
   ├─ 检测人脸位置
   ├─ 质量评估
   └─ 稳定性判断

2. 特征提取
   ├─ 人脸对齐
   ├─ 特征向量计算
   └─ 特征存储

3. 特征比对
   ├─ 加载已知特征
   ├─ 计算相似度
   └─ 判断是否重复

4. 身份核验
   ├─ 与身份证照片比对
   ├─ 相似度计算
   └─ 核验结果
```

## 错误处理

### 异常处理策略

```python
# 业务逻辑层
try:
    result = image_processor.crop_to_spec('一寸')
except ValueError as e:
    # 参数错误
    logger.error(f"参数错误: {e}")
except Exception as e:
    # 其他错误
    logger.error(f"处理错误: {e}")

# 数据访问层
try:
    user = db_helper.add_user(...)
except IntegrityError as e:
    # 数据完整性错误
    logger.error(f"数据错误: {e}")
except Exception as e:
    # 其他错误
    logger.error(f"数据库错误: {e}")
```

## 性能优化

### 缓存策略

```python
# 模型缓存
face_detector = None  # 延迟加载

def get_face_detector():
    global face_detector
    if face_detector is None:
        face_detector = FaceDetection()
    return face_detector
```

### 多线程处理

```python
# 批量处理使用多线程
from concurrent.futures import ThreadPoolExecutor

executor = ThreadPoolExecutor(max_workers=4)
futures = [executor.submit(process_image, img) for img in images]
results = [f.result() for f in futures]
```

### 内存管理

```python
# 及时释放资源
import gc

def process_large_batch():
    for image in images:
        result = process_image(image)
        gc.collect()  # 定期清理内存
```
