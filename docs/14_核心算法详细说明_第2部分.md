## 第三章 磨皮美颜算法

### 3.1 算法背景与意义

磨皮美颜是证件照处理中的重要环节。通过适当的皮肤平滑处理，可以改善照片质量，使人物看起来更加精致。本系统实现了一个基于双边滤波的磨皮算法，同时采用特征保护机制，确保在平滑皮肤的同时保持眼睛、眉毛、嘴巴等重要特征的清晰度。

### 3.2 算法原理详解

#### 3.2.1 双边滤波基础

双边滤波是一种非线性滤波方法，能够在平滑图像的同时保持边界清晰。其核心思想是同时考虑空间距离和像素值差异。

**公式3.1 - 双边滤波**

$$BF(x) = \frac{1}{W_p} \sum_{x_i \in \Omega} f(x_i) \cdot g_s(||x - x_i||) \cdot g_r(|I(x) - I(x_i)|)$$

其中：
- $f(x_i)$ 为像素值
- $g_s$ 为空间高斯核
- $g_r$ 为值域高斯核
- $\Omega$ 为滤波窗口
- $W_p$ 为归一化系数

#### 3.2.2 空间高斯核

**公式3.2 - 空间高斯核**

$$g_s(d) = e^{-\frac{d^2}{2\sigma_s^2}}$$

其中：
- $d$ 为像素之间的欧几里得距离
- $\sigma_s$ 为空间标准差（控制平滑范围）
- 距离越近，权重越大

#### 3.2.3 值域高斯核

**公式3.3 - 值域高斯核**

$$g_r(d) = e^{-\frac{d^2}{2\sigma_r^2}}$$

其中：
- $d$ 为像素值的差异
- $\sigma_r$ 为值域标准差（控制颜色相似度）
- 颜色越相似，权重越大

#### 3.2.4 磨皮强度参数

根据用户设置的磨皮强度（0-1），动态调整滤波参数：

**公式3.4 - 滤波核大小**

$$d = \max(9, \lfloor 15 + 20 \times \text{strength} \rfloor)$$

其中：
- strength ∈ [0, 1]
- d ∈ [9, 35]（像素）
- 强度越高，核越大，平滑效果越明显

**公式3.5 - 颜色标准差**

$$\sigma_{\text{color}} = \max(30, \lfloor 40 + 60 \times \text{strength} \rfloor)$$

其中：
- σ_color ∈ [30, 100]
- 强度越高，颜色相似度要求越低，平滑范围越大

**公式3.6 - 空间标准差**

$$\sigma_{\text{space}} = \max(30, \lfloor 40 + 60 \times \text{strength} \rfloor)$$

其中：
- σ_space ∈ [30, 100]
- 强度越高，平滑范围越大

**公式3.7 - 迭代次数**

$$\text{iterations} = \max(1, \lfloor 1 + \text{strength} \times 2 \rfloor)$$

其中：
- iterations ∈ [1, 3]
- 强度越高，迭代次数越多，平滑效果越明显

#### 3.2.5 特征保护掩膜

为了保护重要特征，创建一个掩膜来标记需要保护的区域：

**公式3.8 - 特征保护掩膜**

$$\text{ProtectMask} = \text{EyeMask} \cup \text{EyebrowMask} \cup \text{MouthMask} \cup \text{NoseMask}$$

其中：
- EyeMask：眼睛周围，半径15像素的圆形
- EyebrowMask：眉毛区域，椭圆形（20×10像素）
- MouthMask：嘴巴区域，椭圆形（20×10像素）
- NoseMask：鼻孔区域，半径8像素的圆形

#### 3.2.6 皮肤区域掩膜

**公式3.9 - 皮肤区域掩膜**

$$\text{SkinMask} = \text{FaceMask} \setminus \text{ProtectMask}$$

其中：
- FaceMask：人脸区域（椭圆形）
- ProtectMask：保护区域
- SkinMask：只在皮肤区域进行磨皮

#### 3.2.7 混合公式

**公式3.10 - 磨皮结果混合**

$$\text{Result}(x, y, c) = \text{Image}(x, y, c) \times (1 - m(x, y) \times b) + \text{Smoothed}(x, y, c) \times m(x, y) \times b$$

其中：
- $m(x, y)$ 为皮肤掩膜（0-1）
- $b$ 为混合强度（0-0.8）
- $c$ 为颜色通道（B、G、R）
- 混合强度越高，磨皮效果越明显

### 3.3 祛痘算法详解

#### 3.3.1 痘痘检测原理

使用形态学操作检测痘痘（暗点）：

**公式3.11 - Top-Hat变换**

$$\text{TopHat} = \text{Image} - \text{Opening}(\text{Image})$$

其中：
- Opening = Erosion(Dilation(Image))
- 用于检测亮点

**公式3.12 - Black-Hat变换**

$$\text{BlackHat} = \text{Closing}(\text{Image}) - \text{Image}$$

其中：
- Closing = Dilation(Erosion(Image))
- 用于检测暗点（痘痘）

**公式3.13 - 痘痘检测**

$$\text{BlemishMap} = \text{TopHat} + \text{BlackHat}$$

#### 3.3.2 痘痘大小过滤

**公式3.14 - 痘痘面积范围**

$$\text{ValidBlemish} = \begin{cases}
\text{True} & \text{if } 2 \leq \text{Area} \leq 50 + 100 \times \text{strength} \\
\text{False} & \text{otherwise}
\end{cases}$$

其中：
- 最小面积：2像素
- 最大面积：50-150像素（取决于强度）

#### 3.3.3 圆度检测

**公式3.15 - 圆度系数**

$$\text{Circularity} = \frac{4\pi \times \text{Area}}{\text{Perimeter}^2}$$

其中：
- Circularity ∈ [0, 1]
- 只保留圆度 > 0.3的区域（痘痘通常比较圆）

#### 3.3.4 Inpaint修复

**公式3.16 - 图像修复**

$$\text{Result} = \text{Inpaint}(\text{Image}, \text{BlemishMask}, r)$$

其中：
- r为修复半径（5-10像素）
- 使用Telea算法进行修复
- 根据周围像素推断修复区域的像素值

### 3.4 眼部增强算法

#### 3.4.1 LAB色彩空间

将RGB色彩空间转换为LAB色彩空间：

**公式3.17 - LAB色彩空间**

$$\begin{cases}
L = \text{亮度通道} \\
a = \text{绿-红色度通道} \\
b = \text{蓝-黄色度通道}
\end{cases}$$

#### 3.4.2 亮度增强

**公式3.18 - 眼部亮度增强**

$$L' = L + m(x, y) \times \text{strength} \times 10$$

其中：
- $m(x, y)$ 为眼睛区域掩膜
- strength ∈ [0, 1]
- 增强量 ∈ [0, 10]

#### 3.4.3 对比度增强

**公式3.19 - 对比度增强**

$$L' = L \times (1 + \text{strength} \times 0.3)$$

其中：
- 对比度增强系数 ∈ [1, 1.3]
- 使眼睛更有神采

### 3.5 唇部增强算法

#### 3.5.1 红色通道增强

**公式3.20 - 唇部红色增强**

$$R' = R + m(x, y) \times \text{strength} \times 30$$

其中：
- $m(x, y)$ 为嘴巴区域掩膜
- strength ∈ [0, 1]
- 增强量 ∈ [0, 30]

### 3.6 牙齿美白算法

#### 3.6.1 亮度增加

**公式3.21 - 牙齿美白**

$$L' = L + m(x, y) \times 20$$

其中：
- $m(x, y)$ 为牙齿区域掩膜
- 固定增加亮度20

**代码3.1 - 磨皮处理实现**

**位置**: `utils/image_helper.py` (第243-330行)

```python
def _skin_smoothing(self, image: np.ndarray, face_mask: np.ndarray,
                   landmarks: Dict) -> np.ndarray:
    """
    智能磨皮，保护重要特征
    
    参数说明：
    - image: 输入图像
    - face_mask: 人脸区域掩膜
    - landmarks: 关键点位置
    
    返回值：
    - 磨皮处理后的图像
    """
    
    # 第一步：创建特征保护掩膜
    protect_mask = self._create_feature_protection_mask(
        image.shape[:2], landmarks)
    
    # 第二步：创建皮肤区域掩膜
    skin_only_mask = cv2.bitwise_and(face_mask, 
                                     cv2.bitwise_not(protect_mask))
    
    # 第三步：获取磨皮强度
    smooth_strength = self.strength_settings['smooth_strength']
    
    # 第四步：计算滤波参数
    d = max(9, int(15 + 20 * smooth_strength))
    sigma_color = max(30, int(40 + 60 * smooth_strength))
    sigma_space = max(30, int(40 + 60 * smooth_strength))
    iterations = max(1, int(1 + smooth_strength * 2))
    
    # 第五步：执行双边滤波
    smoothed = image.copy()
    for i in range(iterations):
        smoothed = cv2.bilateralFilter(smoothed, d, 
                                      sigma_color, sigma_space)
    
    # 第六步：混合结果
    result = image.copy().astype(np.float32)
    mask_norm = skin_only_mask.astype(np.float32) / 255.0
    blend_strength = min(0.8, smooth_strength * 1.0)
    
    for c in range(3):
        result[:, :, c] = (
            image[:, :, c].astype(np.float32) * 
            (1 - mask_norm * blend_strength) +
            smoothed[:, :, c].astype(np.float32) * 
            (mask_norm * blend_strength)
        )
    
    return result.astype(np.uint8)
```

```python
def _blemish_removal(self, image: np.ndarray, 
                    face_mask: np.ndarray) -> np.ndarray:
    """
    祛痘祛瑕疵
    
    参数说明：
    - image: 输入图像
    - face_mask: 人脸区域掩膜
    
    返回值：
    - 祛痘处理后的图像
    """
    
    blemish_strength = self.strength_settings.get('blemish_strength', 0.5)
    
    # 第一步：转换为灰度图 - 第340行
    face_region = cv2.bitwise_and(image, image, mask=face_mask)
    gray_face = cv2.cvtColor(face_region, cv2.COLOR_BGR2GRAY)
    
    # 第二步：形态学检测 - 第343行
    kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (9, 9))
    tophat = cv2.morphologyEx(gray_face, cv2.MORPH_TOPHAT, kernel)
    blackhat = cv2.morphologyEx(gray_face, cv2.MORPH_BLACKHAT, kernel)
    combined = cv2.add(tophat, blackhat)
    
    # 第三步：阈值处理 - 第348行
    threshold_value = int(20 - 15 * blemish_strength)
    threshold_value = max(5, threshold_value)
    _, blemish_mask = cv2.threshold(combined, threshold_value, 
                                    255, cv2.THRESH_BINARY)
    
    # 第四步：过滤大小和圆度 - 第354行
    contours, _ = cv2.findContours(blemish_mask, cv2.RETR_EXTERNAL,
                                   cv2.CHAIN_APPROX_SIMPLE)
    final_mask = np.zeros_like(blemish_mask)
    
    min_area = 2
    max_area = int(50 + 100 * blemish_strength)
    
    for contour in contours:
        area = cv2.contourArea(contour)
        if min_area <= area <= max_area:
            perimeter = cv2.arcLength(contour, True)
            if perimeter > 0:
                circularity = 4 * np.pi * area / (perimeter * perimeter)
                if circularity > 0.3:
                    cv2.fillPoly(final_mask, [contour], 255)
    
    # 第五步：Inpaint修复 - 第370行
    if np.sum(final_mask) > 0:
        inpaint_radius = int(5 + 5 * blemish_strength)
        result = cv2.inpaint(image, final_mask, inpaint_radius,
                            cv2.INPAINT_TELEA)
        return result
    else:
        return image
```
