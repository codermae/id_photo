## 第一章 重复采集检测算法

### 1.1 算法背景与意义

在证件照采集系统中，防止用户重复采集是一个重要的功能需求。重复采集不仅会浪费系统资源，还可能导致数据库中存在大量冗余数据。本系统采用基于人脸特征编码的重复采集检测算法，通过计算人脸特征向量之间的相似度来判断是否为同一个人的重复采集。

### 1.2 算法原理详解

#### 1.2.1 人脸特征编码

人脸特征编码是将人脸图像转换为一个固定维度的特征向量的过程。本系统使用dlib库提供的人脸识别模型，该模型基于深度学习技术，能够将任意人脸图像编码为128维的特征向量。

**特征向量的性质**：
- 维度固定：128维
- 数据类型：浮点数（float32）
- 范围：通常在[-1, 1]之间
- 同一个人的不同照片编码结果相似
- 不同人的编码结果差异大

#### 1.2.2 欧几里得距离度量

欧几里得距离是衡量两个特征向量相似度的标准方法。对于两个128维的特征向量，欧几里得距离定义为：

**公式1.1 - 欧几里得距离**

$$d = \sqrt{\sum_{i=1}^{128} (x_i - y_i)^2}$$

其中：
- $x_i$ 为第一个人脸特征向量的第i个分量
- $y_i$ 为第二个人脸特征向量的第i个分量
- $d$ 为两个特征向量之间的欧几里得距离
- 距离越小，两个人脸越相似

**距离的含义**：
- $d \approx 0$：两个人脸几乎相同（同一个人）
- $0 < d < 0.6$：两个人脸相似度高（可能是同一个人）
- $0.6 \leq d < 1.0$：两个人脸相似度中等（可能是不同的人）
- $d \geq 1.0$：两个人脸差异大（不同的人）

#### 1.2.3 相似度归一化

为了便于理解和使用，将欧几里得距离转换为相似度分数，范围为[0, 1]：

**公式1.2 - 相似度计算**

$$\text{Similarity} = 1 - \min\left(\frac{d}{2}, 1\right)$$

其中：
- $d$ 为欧几里得距离
- 相似度范围为[0, 1]
- 相似度 = 1 表示完全相同
- 相似度 = 0 表示完全不同
- 除以2是为了将距离范围[0, 2]映射到[0, 1]

**相似度的含义**：
- Similarity ≥ 0.6：判定为同一个人（重复采集）
- Similarity < 0.6：判定为不同的人（新采集）

#### 1.2.4 重复判断逻辑

**公式1.3 - 重复采集判断**

$$\text{IsDuplicate} = \begin{cases}
\text{True} & \text{if } \text{Similarity} \geq \text{threshold} \\
\text{False} & \text{otherwise}
\end{cases}$$

其中：
- threshold = 0.6（默认阈值）
- 可根据实际需求调整阈值
- 阈值越高，判定越严格（误识率低，漏识率高）
- 阈值越低，判定越宽松（误识率高，漏识率低）

### 1.3 算法流程

**流程1.1 - 重复采集检测流程**

```
输入：新拍摄的人脸图像 I_new
      用户ID user_id
      
步骤1：人脸检测与特征提取
      - 使用dlib检测图像中的人脸
      - 提取人脸特征向量 E_new (128维)
      - 如果检测失败，返回"无法提取特征"
      
步骤2：从数据库查询已存储特征
      - 根据user_id查询数据库
      - 获取用户的历史人脸特征 E_old (128维)
      - 如果用户首次采集，返回"首次采集"
      
步骤3：计算特征距离
      - 计算欧几里得距离：d = ||E_new - E_old||
      - d的范围通常为[0, 2]
      
步骤4：计算相似度
      - Similarity = 1 - min(d/2, 1)
      - 相似度范围为[0, 1]
      
步骤5：判断是否重复
      - if Similarity >= 0.6:
          返回 "重复采集" + 相似度
      - else:
          返回 "新采集" + 相似度
      
输出：重复检测结果
      {
        'is_duplicate': bool,
        'similarity': float,
        'message': str
      }
```

```python
def check_duplicate_for_user(self, user_id: int, image: np.ndarray) -> Dict:
    """
    检查用户是否已采集过照片
    
    参数说明：
    - user_id: 用户ID（整数）
    - image: 新拍摄的图像（numpy数组，BGR格式）
    
    返回值：
    - is_duplicate: 是否重复（布尔值）
    - similarity: 相似度分数（0-1）
    - message: 检查结果信息（字符串）
    """
    
    # 第一步：获取用户信息
    user = self.db_helper.get_user_by_id(user_id)
    if not user:
        return {'is_duplicate': False, 'similarity': 0.0, 
                'message': '用户不存在'}
    
    # 第二步：检查用户是否首次采集
    if not user.face_encoding:
        return {'is_duplicate': False, 'similarity': 0.0,
                'message': '该用户首次采集'}
    
    # 第三步：提取新图像的人脸特征
    # 使用dlib或face_recognition库
    new_encoding = self.face_manager.encode_face(image)
    if new_encoding is None:
        return {'is_duplicate': False, 'similarity': 0.0,
                'message': '无法提取新图像的人脸特征'}
    
    # 第四步：从数据库获取已存储的特征
    # 特征存储为字节流，需要转换回numpy数组
    existing_encoding = np.frombuffer(user.face_encoding, 
                                      dtype=np.float32)
    
    # 第五步：计算欧几里得距离
    # 公式：d = sqrt(sum((x_i - y_i)^2))
    distance = np.linalg.norm(new_encoding - existing_encoding)
    
    # 第六步：计算相似度
    # 公式：Similarity = 1 - min(d/2, 1)
    similarity = 1.0 - min(distance / 2.0, 1.0)
    
    # 第七步：判断是否重复
    threshold = 0.6  # 相似度阈值
    is_duplicate = similarity >= threshold
    
    # 第八步：返回结果
    result = {
        'is_duplicate': is_duplicate,
        'similarity': similarity,
        'message': f'相似度: {similarity:.2%}'
    }
    
    return result
```

## 第二章 智能裁剪算法

### 2.1 算法背景与意义

证件照的裁剪是一个关键步骤，直接影响最终照片的质量。不同国家和地区对证件照有不同的规格要求。本系统实现了一个智能裁剪算法，能够根据人脸位置自动计算最优的裁剪区域，确保人物居中且符合各种国际标准。

### 2.2 算法原理详解

#### 2.2.1 人脸检测与定位

首先需要检测人脸在图像中的位置和大小：

**公式2.1 - 人脸检测结果**

$$\text{FaceBox} = (x, y, w, h)$$

其中：
- $x$：人脸左上角的x坐标
- $y$：人脸左上角的y坐标
- $w$：人脸的宽度（像素）
- $h$：人脸的高度（像素）

#### 2.2.2 完整头部高度估算

人脸检测通常只能检测到眉毛到下巴的区域，但证件照需要包含头发。因此需要估算完整头部的高度：

**公式2.2 - 头发高度估算**

$$H_{\text{hair}} = 0.4 \times H_{\text{face}}$$

其中：
- $H_{\text{face}}$ 为人脸检测框的高度
- $H_{\text{hair}}$ 为估算的头发高度
- 系数0.4是基于人体比例的经验值

**公式2.3 - 完整头部高度**

$$H_{\text{full}} = H_{\text{face}} + H_{\text{hair}}$$

其中：
- $H_{\text{full}}$ 为从头顶到下巴的完整高度

#### 2.2.3 照片留白计算

根据证件照标准，需要在头顶和下巴处留出适当的空白：

**公式2.4 - 头顶留白**

$$M_{\text{top}} = 0.35 \times H_{\text{full}}$$

其中：
- $M_{\text{top}}$ 为头顶留白高度
- 系数0.35表示留白占完整头部高度的35%

**公式2.5 - 下巴留白**

$$M_{\text{bottom}} = 0.25 \times H_{\text{full}}$$

其中：
- $M_{\text{bottom}}$ 为下巴留白高度
- 系数0.25表示留白占完整头部高度的25%

#### 2.2.4 照片总高度计算

**公式2.6 - 照片总高度**

$$H_{\text{photo}} = H_{\text{full}} + M_{\text{top}} + M_{\text{bottom}}$$

$$H_{\text{photo}} = H_{\text{full}} + 0.35 \times H_{\text{full}} + 0.25 \times H_{\text{full}}$$

$$H_{\text{photo}} = 1.6 \times H_{\text{full}}$$

#### 2.2.5 照片宽度计算

根据目标规格的宽高比计算照片宽度：

**公式2.7 - 照片宽度**

$$W_{\text{photo}} = H_{\text{photo}} \times \text{AspectRatio}$$

其中：
- AspectRatio = $\frac{W_{\text{spec}}}{H_{\text{spec}}}$
- 例如一寸照片：AspectRatio = $\frac{25}{35} \approx 0.714$

#### 2.2.6 裁剪区域计算

**公式2.8 - 裁剪区域左上角坐标**

$$\begin{cases}
x_{\text{crop}} = \max(0, \min(W_{\text{image}} - W_{\text{crop}}, x_{\text{center}} - \frac{W_{\text{crop}}}{2})) \\
y_{\text{crop}} = \max(0, y_{\text{full\_top}} - M_{\text{top}})
\end{cases}$$

其中：
- $x_{\text{center}}$ 为人脸中心的x坐标
- $y_{\text{full\_top}}$ 为完整头部顶部的y坐标
- $(x_{\text{crop}}, y_{\text{crop}})$ 为裁剪区域的左上角坐标
- 确保裁剪区域不超出图像边界

#### 2.2.7 缩放因子计算

如果裁剪区域超出图像边界，需要按比例缩小：

**公式2.9 - 缩放因子**

$$\text{Scale} = \min\left(\frac{W_{\text{image}}}{W_{\text{crop}}}, \frac{H_{\text{image}}}{H_{\text{crop}}}\right) \times 0.95$$

其中：
- 0.95是安全系数，留出5%的余量
- 最终裁剪尺寸 = 原始尺寸 × Scale

#### 2.2.8 最终尺寸调整

**公式2.10 - 最终尺寸调整**

$$\text{FinalImage} = \text{Resize}(\text{CroppedImage}, (W_{\text{spec}}, H_{\text{spec}}))$$

其中：
- $(W_{\text{spec}}, H_{\text{spec}})$ 为目标规格的像素尺寸
- 使用INTER_AREA插值（缩小）或INTER_CUBIC插值（放大）

### 2.4 算法流程

**流程2.1 - 智能裁剪流程**

```
输入：原始图像 I
      目标规格 spec (如"一寸")
      
步骤1：人脸检测
      - 检测人脸位置 (x, y, w, h)
      - 如果未检测到人脸，使用中心裁剪
      
步骤2：估算完整头部
      - H_hair = 0.4 × h
      - H_full = h + H_hair
      - y_full_top = y - H_hair
      
步骤3：计算照片留白
      - M_top = 0.35 × H_full
      - M_bottom = 0.25 × H_full
      - H_photo = H_full + M_top + M_bottom
      
步骤4：计算照片宽度
      - AspectRatio = spec_width / spec_height
      - W_photo = H_photo × AspectRatio
      
步骤5：计算裁剪区域
      - x_crop = center_x - W_photo/2
      - y_crop = y_full_top - M_top
      - 确保不超出图像边界
      
步骤6：执行裁剪
      - cropped = image[y_crop:y_crop+H_photo, 
                        x_crop:x_crop+W_photo]
      
步骤7：调整到目标尺寸
      - if 需要放大:
          final = resize(cropped, (spec_width, spec_height),
                        interpolation=INTER_CUBIC)
      - else:
          final = resize(cropped, (spec_width, spec_height),
                        interpolation=INTER_AREA)
      
输出：符合规格的证件照
```

### 2.5 代码实现详解

**代码2.1 - 智能裁剪实现**

```python
def smart_crop(self, image: np.ndarray, spec: str = '一寸') -> Tuple:
    """
    智能裁剪，保持人物居中且符合证件照标准
    
    参数说明：
    - image: 输入图像（numpy数组，BGR格式）
    - spec: 证件照规格（字符串，如'一寸'）
    
    返回值：
    - 裁剪后的图像
    - 裁剪信息字典
    """
    
    # 获取目标规格信息
    spec_info = self.crop_specs[spec]
    h, w = image.shape[:2]
    
    # 第一步：人脸检测
    face = face_detector.detect_face(image)
    if face is None:
        # 未检测到人脸，使用中心裁剪
        crop_area = self._calculate_center_crop(image.shape, spec_info)
    else:
        x, y, fw, fh = face
        
        # 第二步：估算完整头部高度
        estimated_hair_height = int(fh * 0.4)
        full_head_top = max(0, y - estimated_hair_height)
        full_head_height = fh + estimated_hair_height
        
        # 第三步：计算照片留白
        top_margin = int(full_head_height * 0.35)
        bottom_margin = int(full_head_height * 0.25)
        
        # 第四步：计算照片尺寸
        target_ratio = spec_info['width'] / spec_info['height']
        estimated_photo_height = full_head_height + top_margin + bottom_margin
        estimated_photo_width = int(estimated_photo_height * target_ratio)
        
        # 第五步：计算裁剪区域
        crop_x = max(0, min(w - estimated_photo_width, 
                           x + fw//2 - estimated_photo_width//2))
        crop_y = max(0, full_head_top - top_margin)
        
        crop_area = (crop_x, crop_y, estimated_photo_width, 
                    estimated_photo_height)
    
    # 第六步：执行裁剪
    x, y, w, h = crop_area
    cropped_image = image[y:y+h, x:x+w]
    
    # 第七步：调整到目标尺寸
    target_width = spec_info['width']
    target_height = spec_info['height']
    
    # 选择合适的插值方法
    if cropped_image.shape[1] < target_width:
        # 需要放大
        interpolation = cv2.INTER_CUBIC
    else:
        # 需要缩小
        interpolation = cv2.INTER_AREA
    
    final_image = cv2.resize(cropped_image, 
                            (target_width, target_height),
                            interpolation=interpolation)
    
    return final_image, crop_info
```
