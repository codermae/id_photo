# 图像处理模块

## 模块概述

图像处理模块是证件照系统的核心功能模块，负责对采集到的原始照片进行专业级的处理，包括智能裁剪、背景替换、美颜等操作，最终生成符合规范的标准证件照。

**核心功能**：
- 智能裁剪：基于人脸检测的自动裁剪
- 手动裁剪：用户可拖拽调整裁剪区域
- 背景替换：支持多种背景色的AI智能替换
- 美颜处理：多层次的磨皮和肤色优化
- 参数调整：亮度、对比度、饱和度等实时调整
- 撤销/重做：完整的操作历史管理

## 功能特性

### 1. 智能裁剪 (SmartCropper)

#### 支持的证件照规格

| 规格名称 | 尺寸(mm) | 像素(300DPI) | 用途 |
|---------|---------|------------|------|
| 一寸 | 25×35 | 295×413 | 标准证件照 |
| 小二寸 | 35×49 | 413×579 | 护照 |
| 二寸 | 35×53 | 413×626 | 身份证、驾驶证 |
| 大一寸 | 33×48 | 390×567 | 港澳通行证 |
| 五寸 | 89×127 | 1050×1500 | 大幅照片 |

#### 裁剪算法

```python
# 智能裁剪流程
1. 人脸检测 → 获取人脸位置和大小
2. 计算裁剪区域 → 根据证件照标准定位
3. 人物居中 → 确保人脸在照片中心位置
4. 头部空间 → 保留头顶12%、下巴8%的空间
5. 尺寸调整 → 缩放到目标规格
```

#### 人脸位置标准

- **人脸中心位置**：照片高度的35%处
- **人脸宽度比例**：占照片宽度的60%
- **头顶留白**：照片高度的12%
- **下巴留白**：照片高度的8%

#### 预览和手动调整

- 实时预览裁剪效果
- 支持拖拽调整裁剪框
- 支持缩放调整大小
- 支持旋转调整角度

### 2. 背景替换 (PreciseBackgroundReplacer)

#### 支持的背景色

| 颜色 | RGB值 | 用途 |
|-----|-------|------|
| 白色 | (255, 255, 255) | 标准证件照 |
| 蓝色 | (67, 142, 219) | 护照、签证 |
| 红色 | (255, 0, 0) | 身份证 |
| 浅蓝色 | (173, 216, 230) | 驾驶证 |
| 灰色 | (192, 192, 192) | 其他用途 |

#### 背景替换算法

**优先级顺序**：

1. **AI高精度分割** (rembg + U2-Net)
   - 使用深度学习模型进行人像分割
   - 精度最高，边缘最自然
   - 处理时间：1-2秒

2. **高级肤色检测**
   - 基于HSV颜色空间的肤色检测
   - 支持多肤色范围识别
   - 边缘优化和羽化处理

3. **基础肤色检测** (备用)
   - 简单快速的肤色分割
   - 适用于光线良好的场景

#### 边缘优化

- 形态学操作：腐蚀和膨胀
- 高斯模糊：平滑边缘
- 羽化处理：自然过渡
- 颜色混合：避免生硬边界

### 3. 美颜处理

#### 美颜算法

```python
# 多层次美颜流程
1. 磨皮处理
   - 双边滤波：保留边缘，平滑皮肤
   - 表面模糊：进一步平滑细节
   
2. 肤色优化
   - 肤色均匀化：减少色斑
   - 肤色提亮：增加光泽感
   
3. 对比度调整
   - 自适应对比度增强
   - 保留细节的同时增加立体感
   
4. 锐化处理
   - 轻微锐化增强细节
   - 避免过度锐化导致不自然
```

#### 美颜强度

- **0.0-0.5**：轻度美颜，保留原始质感
- **0.5-1.0**：标准美颜，平衡效果和自然度
- **1.0-1.5**：中度美颜，明显改善肤质
- **1.5-2.0**：强度美颜，显著改善但可能不自然

### 4. 参数调整

#### 支持的调整参数

| 参数 | 范围 | 说明 |
|-----|------|------|
| 亮度 | -100 ~ +100 | 调整整体亮度 |
| 对比度 | -100 ~ +100 | 调整明暗对比 |
| 饱和度 | -100 ~ +100 | 调整色彩饱和度 |
| 锐化 | 0 ~ 100 | 增强细节清晰度 |
| 伽马值 | 0.5 ~ 2.0 | 非线性亮度调整 |

#### 参数预设

- **标准证件照**：推荐参数组合
- **护照照片**：特定的亮度和对比度
- **驾驶证照片**：强调肤色的参数
- **自定义预设**：用户保存的参数组合

### 5. 撤销/重做

- 完整的操作历史记录
- 支持无限次撤销和重做
- 快速切换不同处理步骤
- 与原图对比功能

## 使用示例

### 基本使用流程

```python
from controllers.image_processor import ImageProcessor
from controllers.ai_processor import AIProcessor

# 初始化
ai_processor = AIProcessor()
image_processor = ImageProcessor(ai_processor=ai_processor)

# 1. 加载图像
image_processor.load_image('photo.jpg')

# 2. 智能裁剪
success, crop_info = image_processor.crop_to_spec('一寸')
if success:
    print(f"裁剪成功: {crop_info}")

# 3. 背景替换
image_processor.change_background('白色')

# 4. 美颜处理
image_processor.beautify()

# 5. 参数调整
image_processor.adjust_brightness(10)
image_processor.adjust_contrast(5)

# 6. 保存结果
image_processor.save_processed_image('user_id', '一寸', '白色')

# 7. 撤销操作
image_processor.undo()
```

### 预览功能

```python
# 预览裁剪效果
preview_image, crop_info = image_processor.preview_crop('一寸')

# 预览背景替换
preview_image = image_processor.preview_background('蓝色')

# 预览美颜效果
preview_image = image_processor.preview_beautify(strength=1.5)
```

### 手动裁剪

```python
# 手动裁剪 (x, y, width, height)
crop_area = (100, 50, 300, 400)
image_processor.manual_crop(crop_area, '一寸')
```

## 常见问题

### Q1: 背景替换效果不好怎么办？

**A**: 尝试以下方法：
1. 确保光线充足，人物与背景对比明显
2. 切换不同的替换方法（AI/传统）
3. 手动调整参数
4. 使用手动裁剪精确定位人物

### Q2: 美颜强度如何选择？

**A**: 根据原始照片质量：
- 质量好：0.5-0.8（轻度美颜）
- 质量一般：0.8-1.2（标准美颜）
- 质量差：1.2-1.5（中度美颜）

### Q3: 裁剪后人物位置不对怎么办？

**A**: 使用手动裁剪功能：
1. 点击"手动裁剪"按钮
2. 拖拽调整裁剪框位置
3. 缩放调整大小
4. 确认保存

### Q4: 处理速度太慢怎么办？

**A**: 优化建议：
1. 关闭实时预览
2. 使用传统背景替换方法
3. 降低图像分辨率
4. 关闭美颜处理
