# 07 统计报表系统

## 7.1 技术架构

### 7.1.1 整体方案

**技术选型**: ECharts + QWebEngineView + QThread

**架构图**:
```
┌─────────────────────────────────────┐
│         PyQt5 主线程 (UI)            │
│  - 用户交互                          │
│  - 图表显示 (QWebEngineView)        │
│  - 接收数据信号                      │
└─────────────────────────────────────┘
           ↑ 信号 (data_ready)
┌─────────────────────────────────────┐
│      DataWorker (QThread)           │
│  - 数据库查询                        │
│  - 数据统计计算                      │
│  - ECharts配置生成                   │
└─────────────────────────────────────┘
           ↓ 查询
┌─────────────────────────────────────┐
│         SQLite数据库                 │
│  - User表                           │
│  - Photo表                          │
│  - CollectionRecord表                │
└─────────────────────────────────────┘
```

### 7.1.2 核心类设计

**DataWorker (数据处理线程)**:
```python
class DataWorker(QThread):
    data_ready = pyqtSignal(dict)
    
    def __init__(self, db, collection_id, start_date, end_date):
        super().__init__()
        self.db = db
        self.collection_id = collection_id
        self.start_date = start_date
        self.end_date = end_date
    
    def run(self):
        # 1. 查询数据
        users = self.db.get_all_users(self.collection_id)
        
        # 2. 统计计算
        stats = self.calculate_stats(users)
        
        # 3. 生成图表配置
        charts = self.generate_charts(users, stats)
        
        # 4. 发送信号
        self.data_ready.emit({'stats': stats, 'charts': charts})
```

## 7.2 六大统计图表

### 7.2.1 采集状态分布（环形图）

**用途**: 显示已采集/待采集/失败的比例

**图表类型**: Donut Chart (环形图)

**数据计算**:
```python
def calculate_status_stats(users, start_date, end_date):
    completed = pending = failed = 0
    
    for user in users:
        records = get_records_by_user(user.id)
        records = [r for r in records if start_date <= r.collection_date <= end_date]
        
        if records:
            latest_status = records[-1].status
            if latest_status == 'completed':
                completed += 1
            elif latest_status == 'pending':
                pending += 1
            elif latest_status == 'failed':
                failed += 1
    
    return {
        'completed': completed,
        'pending': pending,
        'failed': failed,
        'rate': (completed / (completed + pending + failed) * 100) if (completed + pending + failed) > 0 else 0
    }
```

**ECharts配置**:

### 7.2.2 地区用户对比（柱状图）

**用途**: 对比不同地区的用户数量

**图表类型**: Bar Chart (柱状图)

**数据计算**:
```python
def calculate_region_stats(users):
    # 身份证前2位 → 省份映射
    id_region_map = {
        '11': '北京市', '12': '天津市', '13': '河北省',
        '31': '上海市', '32': '江苏省', '33': '浙江省',
        '44': '广东省', '51': '四川省', ...
    }
    
    region_stats = {}
    for user in users:
        if user.address:
            # 优先使用地址
            if '省' in user.address:
                region = user.address.split('省')[0] + '省'
            elif '市' in user.address:
                region = user.address.split('市')[0] + '市'
            else:
                region = user.address[:10]
        else:
            # 地址为空，从身份证号推断
            province_code = user.id_number[:2]
            region = id_region_map.get(province_code, '未知')
        
        region_stats[region] = region_stats.get(region, 0) + 1
    
    # 排序取前10
    sorted_regions = sorted(region_stats.items(), key=lambda x: x[1], reverse=True)[:10]
    return sorted_regions
```

```python

    'tooltip': {'trigger': 'axis'},
    'xAxis': {'type': 'category', 'data': regions},
    'yAxis': {'type': 'value'},
    'series': [{
        'type': 'bar',
        'data': values,
        'itemStyle': {'color': '#5470C6'}
    }]
}
```

### 7.2.3 年龄段分布（柱状图）

**用途**: 显示不同年龄段的用户分布

**年龄段划分**:
- 18-25岁
- 26-35岁
- 36-45岁
- 46-55岁
- 56-65岁
- 65岁以上

**数据计算**:
```python
def calculate_age_stats(users):
    age_groups = {
        '18-25': 0, '26-35': 0, '36-45': 0,
        '46-55': 0, '56-65': 0, '65+': 0
    }
    
    today = datetime.now().date()
    for user in users:
        if user.birthday:
            age = (today - user.birthday).days // 365
            if age < 18:
                continue
            elif age <= 25:
                age_groups['18-25'] += 1
            elif age <= 35:
                age_groups['26-35'] += 1
            # ... 其他年龄段
    
    return age_groups
```

### 7.2.4 成功率趋势（折线图）

**用途**: 显示一段时间内的采集成功率变化

**图表类型**: Line Chart (折线图) + Area Style (面积图)

**数据计算**:
```python
def calculate_trend_stats(start_date, end_date, db, collection_id):
    dates = []
    success_rates = []
    
    current_date = start_date
    while current_date <= end_date:
        dates.append(current_date.strftime('%m-%d'))
        
        # 查询当天记录
        records = db.get_records_by_date(current_date, collection_id)
        if records:
            completed = len([r for r in records if r.status == 'completed'])
            total = len([r for r in records if r.status in ['completed', 'pending', 'failed']])
            rate = (completed / total * 100) if total > 0 else 0
        else:
            rate = 0
        
        success_rates.append(rate)
        current_date += timedelta(days=1)
    
    return dates, success_rates
```

**ECharts配置**:
```python

    'dataZoom': [{
        'type': 'inside',

        'end': 100,

        'moveOnMouseMove': False,
        'moveOnMouseWheel': False
    }],
    'xAxis': {
        'type': 'category',
```

### 7.2.6 性别分布（仪表盘）

**用途**: 显示男女比例

**图表类型**: Gauge Chart (仪表盘)

**数据计算**:
```python
def calculate_gender_stats(users):
    gender_stats = {'男': 0, '女': 0, '未知': 0}
    for user in users:
        gender = user.gender if user.gender in ['男', '女'] else '未知'
        gender_stats[gender] += 1
    
    total = sum(gender_stats.values())
    male_ratio = (gender_stats['男'] / total * 100) if total > 0 else 0
    female_ratio = 100 - male_ratio
    
    return male_ratio, female_ratio
```

**ECharts配置**:
```python
{
    'tooltip': {'formatter': '男性占比: {c}%'},
    'series': [{
        'type': 'gauge',
        'startAngle': 180,

        'radius': '90%',
        'min': 0,
        'max': 100,
        'axisLine': {
            'lineStyle': {
                'width': 20,
                'color': [[0.5, '#5470C6'], [1, '#EE6666']]  # 左蓝右红
            }
        },
        'detail': {
            'formatter': f'男性 {male_ratio:.1f}%    女性 {female_ratio:.1f}%',
            'fontSize': 14,
            'offsetCenter': [0, '85%']
```

## 7.3 数据筛选功能

### 7.3.1 筛选条件

**采集任务筛选**:
- 全部采集任务
- 指定采集任务

**日期范围筛选**:
- 开始日期
- 结束日期
- 默认：最近30天

### 7.3.2 筛选逻辑

```python
def filter_data(collection_id, start_date, end_date):
    # 1. 获取用户
    users = db.get_all_users(collection_id)
    
    # 2. 筛选记录
    filtered_records = []
    for user in users:
        records = db.get_records_by_user(user.id)
        records = [r for r in records if start_date <= r.collection_date <= end_date]
        filtered_records.extend(records)
    
    return users, filtered_records
```
