# 功能完善文档

## 概述

本文档记录了证件照系统的四项重要功能完善，提升了系统的实用性和用户体验。

## 1. 统计报表按采集任务分类导出

### 功能描述
优化了统计报表的导出功能，支持按采集任务进行分类统计和导出。

### 主要改进
- **分类导出**: 可选择导出单个采集任务或所有采集任务的报表
- **详细统计**: 包含任务信息、组织机构、完成率等详细数据
- **多工作表**: 为每个采集任务创建独立的用户明细表
- **总计汇总**: 自动计算所有任务的总体统计数据

### 导出内容
#### 单个采集任务报表
- 统计汇总表：任务基本信息和统计数据
- 用户详细表：该任务下所有用户的详细信息

#### 全部采集任务报表
- 采集任务汇总表：所有任务的对比统计
- 各任务用户明细表：每个任务的用户详细信息
- 总计统计：跨任务的汇总数据

### 使用方法
1. 在统计报表页面选择采集任务（或选择"全部"）
2. 设置统计日期范围
3. 点击"导出报表"按钮
4. 选择保存位置，系统自动生成 Excel 文件

## 2. 多规格同时生成功能

### 功能描述
新增多规格同时生成功能，允许用户一次性生成多种规格和背景色的证件照。

### 主要特性
- **批量规格**: 支持同时选择多个证件照规格
- **多种背景**: 支持同时选择多种背景颜色
- **智能组合**: 自动生成所有规格×颜色的组合
- **进度显示**: 实时显示生成进度和当前处理状态
- **快速选择**: 提供"全选"、"常用"、"国际"等快速选择选项

### 支持的规格类别
#### 中国标准
- 一寸、小二寸、二寸、大一寸
- 驾驶证、社保卡等特殊规格

#### 国际标准
- 美国护照、欧盟护照、英国签证
- 日本护照、印度签证、泰国签证等

#### 大尺寸
- 五寸、六寸照片

### 使用方法
1. 加载图像后点击"🎯 多规格同时生成"
2. 选择需要的证件照规格（支持多选）
3. 选择需要的背景颜色（支持多选）
4. 设置处理选项（美颜、处理模式等）
5. 点击"🚀 开始生成"，等待批量处理完成

### 生成结果
- 自动保存到指定目录
- 文件名包含规格和背景色信息
- 显示生成统计和文件列表

## 3. 原图对比预览功能

### 功能描述
在图像处理界面添加原图对比功能，方便用户查看处理前后的效果差异。

### 主要特性
- **一键对比**: 点击"📷 与原图对比"按钮切换对比模式
- **并排显示**: 左侧显示处理后图像，右侧显示原始图像
- **清晰标识**: 明确标注"处理后"和"原图"
- **状态提示**: 实时显示当前预览模式
- **快速切换**: 可随时开启或关闭对比模式

### 界面布局
#### 普通模式
- 单一预览窗口显示处理后图像
- 状态显示："当前预览: 处理后图像"

#### 对比模式
- 双预览窗口并排显示
- 左侧：处理后图像（蓝色标签）
- 右侧：原始图像（绿色标签）
- 状态显示："对比模式: 左侧为处理后，右侧为原图"

### 使用方法
1. 加载图像后，系统自动保存原图
2. 进行任意图像处理操作
3. 点击"📷 与原图对比"查看效果差异
4. 再次点击可关闭对比模式

## 使用指南

### 多规格生成最佳实践
1. **规格选择**: 根据实际需求选择规格，避免生成过多不必要的文件
2. **背景色选择**: 优先选择目标用途要求的标准背景色
3. **处理模式**: 推荐使用"高保真模式"获得最佳效果
4. **批量大小**: 单次生成建议不超过50张（10规格×5颜色）

### 原图对比使用技巧
1. **处理前对比**: 在开始处理前先查看原图质量
2. **实时对比**: 每次处理后都可以对比查看效果
3. **细节检查**: 重点关注人脸、头发、边缘等关键区域
4. **效果评估**: 通过对比决定是否需要调整处理参数

### 报表导出建议
1. **定期导出**: 建议每周或每月定期导出统计报表
2. **分类管理**: 大型组织建议按采集任务分别导出
3. **数据备份**: 导出的报表可作为重要的数据备份
4. **分析利用**: 利用Excel的数据分析功能进行深度统计
