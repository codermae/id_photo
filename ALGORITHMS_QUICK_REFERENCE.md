## 人脸识别方案 (2个)

### 1. dlib (主方案)
- **库**: face-recognition
- **特征**: 128 维向量
- **相似度**: 欧几里得距离
- **阈值**: 60%

### 2. InsightFace (可选)
- **特征**: 512 维向量
- **相似度**: 余弦相似度
- **精度**: 更高

---

## 传统算法

### 图像处理
- **形态学**: 腐蚀、膨胀、闭运算、开运算
- **滤波**: 高斯模糊、双边滤波、Unsharp Mask
- **边缘检测**: Canny 边缘检测
- **色彩处理**: HSV 调整、Gamma 校正

### 质量评估
- **人脸检测置信度**
- **人脸大小比例**
- **光线条件分析**
- **清晰度评估** (Laplacian 方差)
- **姿态估计**

### 智能裁剪
- 人脸检测 → 头部高度估算 → 比例计算 → 边距调整 → 缩放

---

## 处理流程

### 采集端
```
摄像头 → fdlite 人脸检测 → 质量评估 → 身份核验 → 保存
```

### 处理端
```
原始图片 → 智能裁剪 → U2-Net 抠图 → 背景替换 → 
GFPGAN 增强 → 美颜处理 → 参数调整 → 输出
```

### 批量处理
```
多张图片 × 多规格 × 多颜色 → 队列处理 → 批量输出
```

## 快速查询

**问**: 用了什么深度学习模型？
**答**: U2-Net (抠图), GFPGAN (增强), InsightFace (检测识别)

**问**: 人脸识别用的什么算法？
**答**: dlib CNN (128维) 或 InsightFace (512维)

**问**: 背景替换怎么做的？
**答**: U2-Net 生成 mask + 传统 CV 优化边缘 + 合成背景

**问**: 处理速度怎么样？
**答**: 单张 3-5 秒，批量后台处理，不阻塞 UI

**问**: 支持 GPU 吗？
**答**: 当前 CPU，可选配置 CUDA 加速
