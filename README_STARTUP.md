## 🛠️ 推荐启动方式

### 方式1：简化启动（推荐）
```bash
python start_clean.py
```
- 输出更简洁
- 自动抑制警告
- 显示启动进度

### 方式2：标准启动
```bash
python main.py
```
- 显示详细调试信息
- 适合开发调试

## 🔧 故障排除

### 如果遇到 NumPy 兼容性问题
```bash
python fix_dependencies.py
```

### 如果遇到依赖缺失
```bash
pip install -r requirements.txt
```

### 如果摄像头功能异常
- 检查摄像头是否被其他程序占用
- 确保摄像头驱动正常
- 摄像头功能是可选的，不影响图像处理

## 📋 功能验证

启动后可以验证以下功能：

### 1. 基础功能
- ✅ 图像加载和显示
- ✅ 证件照规格裁剪
- ✅ 背景替换
- ✅ 统计报表

### 2. 新增功能
- ✅ 多规格同时生成
- ✅ 原图对比预览
- ✅ 全球证件照规格（22种）
- ✅ 分类统计报表导出

### 3. 高级功能（可选）
- ⚠️ 高保真模式（需要 InsightFace）
- ⚠️ 人脸识别（需要 face_recognition）
- ⚠️ 摄像头采集（需要摄像头设备）

## 💡 使用建议

1. **首次使用**：建议先用"图像处理"功能测试基本流程
2. **批量处理**：使用"多规格同时生成"提高效率
3. **效果对比**：使用"原图对比"功能查看处理效果
4. **数据管理**：定期导出统计报表进行数据分析

启动成功后，可以按以下步骤测试：
