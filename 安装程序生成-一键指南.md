## ⚠️ 如果出现问题

### 问题1：找不到NSIS

**症状**：显示"未找到NSIS编译器"  
**解决方案**：
1. 下载NSIS：https://nsis.sourceforge.io/
2. 安装到默认位置
3. 重新运行 `python pack.py`

**或者**（不装NSIS也能用）：
- 使用绿色版本 `证件照采集系统.exe`
- 用户可以直接运行，也能使用

### 问题2：PyInstaller未安装

**症状**：显示"PyInstaller未安装"  
**解决方案**：脚本会自动安装，稍等一下

### 问题3：文件太大

**情况**：生成的文件 > 300MB  
**原因**：包含了所有依赖库  
**解决**：这是正常的。如果想更小：
- 使用 `build_simple.py` 生成更小的版本

### 问题4：其他错误

检查错误信息，常见原因：
- 项目文件缺失 → 检查 `config/`, `views/`, `controllers/` 目录
- 模块导入错误 → 运行 `pip install -r requirements.txt`
- 权限问题 → 以管理员身份运行命令行

## 💡 常见问题

**Q：用户需要安装Python吗？**  
A：不需要！.exe文件已包含完整的Python运行时。

**Q：安装到哪里？**  
A：默认：`C:\Program Files\ID Photo\证件照采集系统`  
用户可以在安装向导中更改。

**Q：用户的数据在哪？**  
A：`%APPDATA%\证件照采集系统\data`（用户目录）

**Q：如何卸载？**  
A：打开 `控制面板` → `程序和功能` → 找到应用 → `卸载`

**Q：如何升级版本？**  
A：修改 `pack.py` 中的 `VERSION = "1.0.0"` 改成 `"1.0.1"`  
重新运行 `python pack.py`  
新版本可以覆盖安装，用户数据自动保留。

**Q：能添加应用图标吗？**  
A：可以。将 `app.ico` 放在项目根目录，脚本会自动识别。

**Q：能在其他操作系统上运行吗？**  
A：不行。这个方案只针对Windows。
