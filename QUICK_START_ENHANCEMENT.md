## 🚀 快速实现示例

### 示例 1：添加主色按钮

```python
# 定义颜色常量
PRIMARY_COLOR = "#0078D4"

# 创建样式表
BUTTON_STYLE = f"""
    QPushButton {{
        background-color: {PRIMARY_COLOR};
        color: white;
        border: none;
        border-radius: 4px;
        padding: 8px 16px;
        font-size: 11pt;
        font-weight: bold;
    }}
    QPushButton:hover {{
        background-color: #005A9E;
    }}
    QPushButton:pressed {{
        background-color: #003D7A;
    }}
    QPushButton:disabled {{
        background-color: #D0D0D0;
        color: #999999;
    }}
"""

# 应用到按钮
button = QPushButton("保存")
button.setStyleSheet(BUTTON_STYLE)
```

### 示例 2：改进错误提示

```python
# 优化前
QMessageBox.critical(self, "错误", "采集失败")

# 优化后
error_message = """
✗ 采集失败

原因：摄像头连接失败
建议：请检查摄像头连接或重启应用
错误代码：ERR_CAMERA_001
"""
QMessageBox.critical(self, "采集失败", error_message)
```

### 示例 3：创建简单图表

```python
from views.chart_widget import ChartWidget

# 创建图表
chart = ChartWidget()

# 设置图表选项
option = {
    'title': {'text': '采集完成率'},
    'series': [{
        'type': 'gauge',
        'data': [{'value': 80, 'name': '完成率'}]
    }]
}

# 显示图表
chart.set_option(option)
```
