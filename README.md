## 快速开始

```bash
# 1. 进入项目目录
cd id_photo_system

# 2. 激活虚拟环境（如果已创建）
.\venv\Scripts\activate  # Windows

# 3. 安装依赖（首次运行）
pip install -r requirements.txt

# 4. 启动程序
python main.py
# 或
python START_NOW.py
```

## 主要功能说明

### 1. 身份证读取
- 支持身份证读卡器
- 自动提取姓名、身份证号、照片等信息
- 防止重复保存（检查身份证号）
- 保存后自动清空用户选择

### 2. 摄像头拍照
- 实时预览（640x480优化分辨率）
- 人脸检测和质量评分
- 身份核验（与身份证照片对比）
- 相似度持久化显示

### 3. 图像处理
**裁剪规格**: 一寸、二寸、小二寸、大一寸、五寸

**背景替换**:
- 精细模式：AI+边缘优化，处理头发丝

**美颜功能**:
- 磨皮（可调强度0-1）
- 祛痘（可调强度0-1，保护眼睛/眉毛/嘴巴）

**调整参数**:
- 亮度调节（-100到+100）
- 对比度调节（0.5到2.0）

**手动裁剪**: 支持自定义裁剪框

### 4. 批量处理
- 批量导入图片（支持文件夹）
- 统一处理参数
- 实时进度显示
- 错误处理和统计

## 常见问题

### 1. 摄像头打不开
- 检查摄像头是否被其他程序占用
- 尝试修改config/config.py中的CAMERA_INDEX
- 检查摄像头驱动是否正常

### 2. 背景替换效果不好
- 使用"精细模式"处理头发丝
- 确保光线充足，背景简单
- 避免复杂背景和阴影

### 3. 祛痘没效果
- 提高祛痘强度（0.7-1.0）
- 确保人脸检测成功
- 检查图片质量和分辨率

### 4. 批量处理速度慢
- 精细模式处理时间较长，如需快速处理可考虑降低图片质量
- 减少美颜选项
- 关闭不必要的参数调整
