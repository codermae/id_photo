# 🎉 多规格批量生成 - 最终融合版

## 💡 核心理念

**一个功能，两种用法：**
- 可以选择 **1张图片** → 生成多种规格和颜色
- 可以选择 **多张图片** → 每张都生成多种规格和颜色

## 🚀 使用方式

### 打开功能
```
图像处理界面 → [🚀 多规格批量生成]
```

### 操作步骤

#### 1. 选择图片
- **单张**: 点击"选择图片文件"，选择1张图片
- **多张**: 点击"选择文件夹"，选择包含多张图片的文件夹

#### 2. 选择规格
- 勾选需要的证件照规格
- 使用快速按钮：
  - **全选**: 10种不同尺寸
  - **常用**: 4种常用规格
  - **国际**: 3种国际规格

#### 3. 选择颜色
- 勾选需要的背景颜色
- 默认选中：白色、蓝色、美国护照蓝、欧盟护照灰

#### 4. 查看生成数量
- 界面会实时显示将生成的照片总数
- 显示预计时间和存储空间

#### 5. 配置处理选项
- 背景模式：精细模式 或 高保真模式
- Alpha Matte：提升边缘质量
- 美颜：启用美颜处理
- 亮度：调整亮度值

#### 6. 选择输出目录
- 点击"选择"按钮
- 选择处理后照片的保存位置

#### 7. 开始生成
- 点击"▶️ 开始生成"
- 确认生成信息
- 等待处理完成

## 📊 使用示例

### 示例1: 单张多规格
```
输入: 1张图片
规格: 4种 (一寸、小二寸、二寸、大一寸)
颜色: 4种 (白色、蓝色、红色、灰色)
输出: 1 × 4 × 4 = 16张照片
时间: ~48秒
```

### 示例2: 多张批量
```
输入: 100张图片
规格: 4种 (一寸、小二寸、二寸、大一寸)
颜色: 4种 (白色、蓝色、红色、灰色)
输出: 100 × 4 × 4 = 1600张照片
时间: ~80分钟
```

### 示例3: 完整覆盖
```
输入: 100张图片
规格: 10种 (全部不同尺寸)
颜色: 17种 (全部背景色)
输出: 100 × 10 × 17 = 17000张照片
时间: ~850分钟
```
