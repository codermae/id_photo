# 直接生成更新说明

## 更新内容

已移除"应用处理"按钮的二次确认对话框，用户勾选好规格和颜色后，点击"应用处理"将直接开始生成，无需再次确认。

## 工作流程

1. 用户在界面上勾选想要的规格（可多选）
2. 用户勾选想要的背景色（可多选）
3. 点击"应用处理"按钮
4. 系统直接开始生成，显示进度条
5. 生成完成后显示结果

## 逻辑判断

- **单规格 + 单颜色**: 直接处理当前图像并显示
- **多规格 或 多颜色**: 自动调用 `generate_multi_spec()` 批量生成

## 代码变更

### 文件: `id_photo_system/views/process_view.py`

移除了 `generate_multi_spec()` 方法中的确认对话框：

```python
# 移除的代码：
# 确认生成
reply = QMessageBox.question(
    self, "确认生成", 
    f"将生成 {len(selected_specs)} 规格 × {len(selected_colors)} 颜色 = {total} 张照片\n\n确定要继续吗？",
    QMessageBox.Yes | QMessageBox.No
)

if reply != QMessageBox.Yes:
    return
```

现在用户勾选后直接点击"应用处理"即可开始生成，无需二次确认。
